**Основные концепции**

В этом разделе мы рассмотрим основные концепции чисел в Python, включая базовые числовые типы, преобразование типов, перегрузку операторов и полиморфизм.

**Базовые числовые типы**

Python имеет несколько базовых числовых типов: int (целое число), float (вещественное число) и complex (комплексное число). Эти типы можно использовать для хранения и манипулирования числами.

В Python 3.0 существует возможность работать с длинными целыми числами, которые не ограничены размером памяти. Это означает, что вы можете работать с очень большими целыми числами без необходимости использовать специальный синтаксис.

**Преобразование типов**

Python автоматически выполняет преобразование типов при смешивании числовых типов в выражении. Например, если вы попытаетесь сложить строку и число, Python будет выдать ошибку, если вы не выполните преобразование типа одного из операндов.

В Python 2.6 допускается сравнивание разнотипных нечисловых значений, но при этом преобразование типов операндов не выполняется. В версии 3.0 сравнивание разнотипных нечисловых значений не допускается и приводит к исключению.

**Перегрузка операторов**

Python позволяет перегружать любые операторы с помощью классов или расширений на языке C для работы с создаваемыми объектами. Например, объекты, реализованные в виде классов, могут участвовать в операциях сложения, индексироваться с помощью выражения [i] и так далее.

Python автоматически перегружает некоторые операторы, чтобы с их помощью можно было выполнять различные действия, в зависимости от типа встроенных объектов. Например, оператор + выполняет операцию сложения, когда применяется к числам, но когда он применяется к последовательностям, таким как строки или списки, он выполняет операцию конкатенации.

**Полиморфизм**

Полиморфизм — это особенность Python, которая позволяет выполняемую операция зависеть от типов объектов-операндов, над которыми она выполняется. Например, оператор + может выполнять любые действия, когда применяется к объектам, которые вы определяете с помощью классов.

**Форматы представления чисел**

Python поддерживает несколько форматов представления чисел: десятичная, шестнадцатеричная и двоичная. Вы можете использовать литералы для создания целых чисел в этих форматах.

Например, вы можете создать целое число в шестнадцатеричной форме с помощью литерала 0x10 или в двоичной форме с помощью литерала 0b101. Вы также можете использовать функции oct и hex для преобразования десятичных чисел в восьмеричное и шестнадцатеричное представления соответственно.

**Битовые операции**

Python поддерживает битовые операции, такие как сдвиг влево (<<), сдвиг вправо (>>), побитовое ИЛИ (|) и побитовое И (&). Вы можете использовать эти операции для манипулирования отдельными битом.

Например, вы можете сдвинуть двоичное значение 1 на две позиции влево с помощью операции x << 2. Вы также можете выполнить побитовое ИЛИ между двумя двоичными значениями с помощью операции x | y.

**Встроенные функции**

Python предоставляет несколько встроенных функций для работы с числами, такие как pow и abs. Вы можете использовать эти функции для вычисления степени и абсолютного значения числа соответственно.

Например, вы можете вычислить степень 2^4 с помощью функции pow(2, 4). Вы также можете получить абсолютное значение -42.0 с помощью функции abs(-42.0).

**Модуль math**

Python предоставляет модуль math для работы с числами. Вы можете использовать функции из этого модуля для вычисления математических констант и функций, таких как sin и cos.

Например, вы можете получить значение pi с помощью функции math.pi. Вы также можете вычислить синус 2*π/180 с помощью функции math.sin(2 * math.pi / 180).
