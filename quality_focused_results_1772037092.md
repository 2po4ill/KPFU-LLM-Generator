## Основные концепции

**Основные концепции: Работа со строками**

**Часть 1: Работа со строками**

В этой части мы рассмотрим основные концепции работы с строками в языке Python.

### 1. Строковые литералы

Строковые литералы — это константы, представляющие строки текста. Они обозначаются двумя кавычками и могут содержать любые символы, включая пробелы и специальные символы. Например:

```python
"Hello, World!"
'Hello, World!'
```

Строковые литералы можно использовать в программе напрямую или хранить их в переменных.

### 2. Методы строк

Методы строк — это функции, которые можно вызывать на строковых объектах. Они позволяют выполнять различные операции с строками, такие как поиск подстрок, замена символов и т. д. Например:

```python
s = "Hello, World!"
print(s.upper())  # Выводит: HELLO, WORLD!
```

Методы строк можно найти в документации языка Python.

### 3. Форматирование

Форматирование — это способ создания строки с использованием специальных кодов для форматирования данных. Например:

```python
name = "John"
age = 30
print(f"Мое имя: {name}, мой возраст: {age}")
```

В этом примере мы используем функцию `f-строки` (formatted string literals) для создания строки с включением переменных.

### 4. Срезы строк

Срезы строк — это способ получить часть строки, указав начальный и конечный индексы. Например:

```python
s = "Hello, World!"
print(s[0:5])  # Выводит: Hello
```

В этом примере мы получаем первые пять символов строки.

### 5. Escape-последовательности

Escape-последовательности — это специальные последовательности символов, которые позволяют включать в строку символы, которые иначе бы были интерпретированы как управляющие последовательности или разделители строк. Например:

```python
s = "Hello\nWorld!"
print(s)
```

В этом примере мы используем escape-последовательность `\n` для включения символа новой строки.

Эти концепции являются основными в работе с строками в языке Python и позволяют выполнять различные операции с текстом.

В этой части мы рассмотрим четыре важных концепции, связанные с работой в Python: работа со списками, работа со словарями, работа с файлами, итераторы и генераторы.

### 1. Работа со списками

**Списки (Lists)** - это тип данных в Python, представляющий собой коллекцию элементов, которые могут быть любого типа данных. Списки хранятся в квадратных скобках `[]` и разделяются запятыми `,`.

**Создание списка:**

- Используем квадратные скобки `[]`: `[1, 2, 3]`
- Используем функцию `list()`: `list([1, 2, 3])`

**Операции с списками:**

- **Индексация**: доступ к элементам списка по индексу (нумерации): `my_list[0]` - первый элемент
- **Срезы**: получение части списка: `my_list[1:3]` - элементы со второго по третий
- **Длина**: количество элементов в списке: `len(my_list)`
- **Добавление/Удаление элемента**: `append()`, `extend()`, `insert()`, `remove()`, `pop()`

**Примеры использования списков:**

```python
my_list = [1, 2, 3]
print(my_list[0])  # Выводит первый элемент списка
print(len(my_list))  # Выводит длину списка

# Добавление элемента в конец списка
my_list.append(4)
print(my_list)  # [1, 2, 3, 4]

# Удаление первого элемента
my_list.pop(0)
print(my_list)  # [2, 3, 4]
```

### 2. Работа со словарями

**Словари (Dictionaries)** - это тип данных в Python, представляющий собой коллекцию пар ключ-значение, где каждый ключ уникален и соответствует определенному значению.

**Создание словаря:**

- Используем фигурные скобки `{}`: `{'name': 'John', 'age': 30}`
- Используем функцию `dict()`: `dict(name='John', age=30)`

**Операции с словарями:**

- **Доступ к значению**: доступ к значению по ключу: `my_dict['name']`
- **Добавление/Удаление пары ключ-значение**: `update()`, `clear()`, `pop()`
- **Проверка наличия ключа**: `in`

**Примеры использования словарей:**

```python
my_dict = {'name': 'John', 'age': 30}
print(my_dict['name'])  # Выводит значение по ключу 'name'

# Добавление новой пары ключ-значение
my_dict.update({'city': 'New York'})
print(my_dict)  # {'name': 'John', 'age': 30, 'city': 'New York'}

# Удаление пары ключ-значение
my_dict.pop('age')
print(my_dict)  # {'name': 'John', 'city': 'New York'}
```

### 3. Работа с файлами

**Файлы (Files)** - это источник данных, который может хранить данные в виде текста или бинарных данных.

**Режимы открытия файла:**

- `r` - чтение
- `w` - запись
- `a` - добавление
- `x` - создание нового файла

**Функции работы с файлами:**

- `open()`: открывает файл в заданном режиме
- `read()`, `write()`, `close()`: операции чтения и записи данных в файл

**Примеры использования файлов:**

```python
# Открытие файла для записи
file = open('example.txt', 'w')
file.write('Hello, world!')
file.close()

# Открытие файла для чтения
file = open('example.txt', 'r')
print(file.read())
file.close()
```

### 4. Итераторы и генераторы

**Итераторы (Iterators)** - это объекты, которые позволяют перебирать коллекции данных.

**Генераторы (Generators)** - это функции, которые возвращают итератор для последовательности данных.

**Примеры использования итераторов и генераторов:**

```python
# Итератор для списка
my_list = [1, 2, 3]
my_iter = iter(my_list)
print(next(my_iter))  # Выводит первый элемент

# Генератор для последовательности чисел
def my_generator():
    for i in range(10):
        yield i

gen = my_generator()
print(next(gen))  # Выводит первое число в последовательности
```
