**Основные концепции: Работа со строками**

**Часть 1: Работа со строками**

В этой части мы рассмотрим четыре важных концепции работы с строками в языке Python: строковые литералы, методы строк, форматирование и срезы строк.

### 1. Строковые литералы

Строковые литералы — это константы, представляющие строки текста. Они определяются двумя кавычками и могут содержать любые символы, включая пробелы, цифры и специальные символы.

Например:

```python
строка = "Привет, мир!"
```

В этом примере строка «Привет, мир!» является строковым литералом. Строковые литералы можно использовать в программе как константы, не подверженные изменениям.

### 2. Методы строк

Методы строк — это функции, которые можно вызывать на строках для выполнения различных операций. Например, метод `upper()` преобразует строку в верхний регистр, а метод `lower()` — в нижний регистр.

Например:

```python
строка = "Привет, мир!"
print(строка.upper())  # Выводит: ПРИВЕТ, МИР!
print(строка.lower())  # Выводит: привет, мир!
```

Методы строк можно использовать для выполнения различных операций с строками, таких как поиск и замена подстрок, вычисление длины строки и т. д.

### 3. Форматирование

Форматирование — это способ представления данных в определенной форме. В языке Python можно использовать функцию `format()` для форматирования строк.

Например:

```python
имя = "Иван"
возраст = 25
print("Меня зовут {} и мне {} лет.".format(имя, возраст))  # Выводит: Меня зовут Иван и мне 25 лет.
```

Функция `format()` позволяет указывать позиции в строке для замены на фактические значения.

### 4. Срезы строк

Срезы строк — это способ получить часть строки, начиная с определенного индекса и заканчивая другим индексом. В языке Python можно использовать квадратные скобки `[]` для получения среза строки.

Например:

```python
строка = "Привет, мир!"
print(строка[0:5])  # Выводит: Приве
```

В этом примере строка «Привет, мир!» является исходной строкой. Срез `строка[0:5]` возвращает часть строки от индекса 0 до индекса 5 (включительно).

### Escape-последовательности

Escape-последовательности — это специальные последовательности символов, которые начинаются с обратного слеша (`\`). Они используются для представления спецсимволов или для указания того, что следующий символ должен быть интерпретирован как обычный символ, а не как часть команды.

Например:

* `\n` — новая строка
* `\\` — обратный слеш
* `\t` — горизонтальная табуляция

### Строки в Python

В языке Python строки представляют собой последовательность символов, заключенных в кавычки. Существуют два типа строк: строка и байтовая строка.

* `str` — строка (представляет собой последовательность Unicode-символов)
* `bytes` — байтовая строка (представляет собой последовательность байт)

Например:

```python
строка = "Hello, World!"
байтовая_строка = b"Hello, World!"
```

### Работа со строками

В языке Python существует ряд методов и операций для работы со строками.

* Индексация: доступ к конкретному символу строки по его индексу.
* Срезы: получение подстроки из исходной строки.
* Методы строк: `upper()`, `lower()`, `strip()`, `replace()` и т.п.

Например:

```python
строка = "Hello, World!"
print(строка[0])  # Выводит первый символ строки
print(строка[:5])  # Выводит первые пять символов строки
print(строка.upper())  # Возвращает строку в верхнем регистре
```

### Операции над строками

В языке Python существует ряд операций для работы со строками.

* Сложение: объединение двух строк.
* Вычитание: получение подстроки из исходной строки.
* Умножение: повторение строки определенное количество раз.

Например:

```python
строка1 = "Hello"
строка2 = "World!"
print(строка1 + строка2)  # Выводит объединенную строку
print(строка1 * 3)  # Возвращает строку, повторенную три раза
```

### Строчные функции

В языке Python существует ряд функций для работы со строками.

* `len()`: возвращает длину строки.
* `type()`: возвращает тип строки (str или bytes).
* `format()`: форматирует строку по шаблону.

Например:

```python
строка = "Hello, World!"
print(len(строка))  # Выводит длину строки
print(type(строка))  # Возвращает тип строки
print("My name is {name}".format(name="John"))  # Форматирует строку по шаблону
```
