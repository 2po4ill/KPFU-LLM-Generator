**Методы форматирования строк**

Python предоставляет два основных способа форматирования строк: оператор % и метод format. Оператор % используется для форматирования строки, используя спецификаторы %s, %d, %f и т. д.

Например:
```
>>> name = 'John'
>>> age = 25
>>> print('Hello, my name is %s and I am %d years old.' % (name, age))
```

Метод format используется для форматирования строки, используя синтаксис {0}, {1} и т. д.

Например:
```
>>> name = 'John'
>>> age = 25
>>> print('Hello, my name is {} and I am {} years old.'.format(name, age))
```

**Сравнение строк**

В Python строки можно сравнивать с помощью операторов == и !=. Например:
```
>>> s1 = 'hello'
>>> s2 = 'world'
>>> s1 == s2
False
>>> s1 != s2
True
```

**Сортировка**

В Python строки можно сортировать с помощью метода sort(). Например:
```
>>> s = ['apple', 'banana', 'cherry']
>>> s.sort()
>>> s

```


**Списки как последовательности**

Списки являются последовательностями, то есть они имеют определенный порядок элементов. Это означает, что каждый элемент списка имеет индекс или номер позиции в списке. Индексы начинаются с нуля и увеличиваются на единицу для каждого следующего элемента.

**Операции над списками**

Списки поддерживают ряд операций, которые можно использовать для манипулирования их содержимым. Эти операции включают:

* **Извлечение среза**: извлечение части списка, начиная с определенного индекса и заканчивая другим индексом.

**Присваивание по индексам и срезам**

Списки поддерживают присваивание по индексам и срезам. Это означает, что можно изменить содержимое списка, присвоив новое значение элементу или части списка.

* **Присваивание по индексу**: присвоение нового значения элементу в списке по его индексу.
* **Присваивание среза**: присвоение нового значения части списка, начиная с определенного индекса и заканчивая другим индексом.

**Сравнение операторов % и метода format**

В языке Python существует два способа форматирования строк: с помощью оператора `%` и метода `format`. Оба этих способа можно использовать для форматирования строк, но есть некоторые различия между ними. Оператор `%` является более старым способом форматирования, а метод `format` был добавлен в Python 2.6.

Один из главных преимуществ метода `format` над оператором `%` заключается в том, что он позволяет использовать универсальные аргументы. Универсальные аргументы — это переменные, которые можно использовать в выражении форматирования, а не только конкретные значения. Например:

```
>>> '{0:.2f}, {1:.2f}'.format(3.14159, 3.14159)
'3.14, 3.14'
```

В этом примере мы используем два универсальных аргумента: `{0}` и `{1}`, которые соответствуют двум значениям в методе `format`.

Однако есть и недостатки использования метода `format`. Например, он может быть менее читаемым, чем оператор `%`, особенно при работе с длинными выражениями форматирования. Кроме того, метод `format` не поддерживает некоторые функции, которые доступны в операторе `%`.

**Использование универсальных аргументов**

Универсальные аргументы — это одна из наиболее мощных функций метода `format`. Они позволяют использовать переменные в выражении форматирования, а не только конкретные значения. Например:

```
>>> '{0} + {1} = {2}'.format(3, 4, 7)
'3 + 4 = 7'
```

В этом примере мы используем три универсальных аргумента: `{0}`, `{1}` и `{2}`, которые соответствуют трем значениям в методе `format`.

**Примечания к возможному отказу от использования оператора %**

В будущем разработчики языка Python могут объявить оператор `%` нерекомендуемым, в пользу метода `format`. Это может произойти из-за того, что метод `format` является более мощным и гибким, чем оператор `%`.

Однако это не означает, что оператор `%` будет удален из языка. Он все еще будет доступ
