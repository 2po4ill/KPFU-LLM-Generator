**Работа со строками в языке Python**

В языке Python строки являются одной из основных типов данных. Они представляют собой последовательность символов, заключенных в кавычки или апострофы.

**Кортежи (Tuples)**

Кортеж — это неизменяемый список элементов, заключенный в круглые скобки. Кортежи используются для хранения и передачи набора данных, который не должен изменяться.

Пример:
```python
my_tuple = ("apple", "banana", "cherry")
print(my_tuple[0])  # выводит "apple"
```
**Файлы (Files)**

В языке Python файлы представляют собой поток байт, которые можно читать и записывать. Файлы используются для хранения данных на диске.

Пример:
```python
file = open("example.txt", "r")
print(file.read())
file.close()
```
**Другие базовые типы**

В языке Python есть еще несколько базовых типов:

* **Целые числа (Integers)**: представляют собой целые числа, например 1, 2, 3 и т. д.
* **Соотношения (Floats)**: представляют собой десятичные числа, например 3.14 или -0.5
* **Логические значения (Booleans)**: представляют собой логические значения true или false

Пример:
```python
x = 5  # целое число

```
**Введение в типы объектов языка Python**

**Операции с строками**

В языке Python существует несколько операций, которые можно использовать для работы со строками:

* **Вычитание строк**: позволяет вычесть одну строку из другой.
* **Умножение строки**: позволяет повторить строку определенное количество раз.

