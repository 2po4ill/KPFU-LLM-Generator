## Основные концепции

**Основные концепции: Работа со строками**

### Строковые литералы

Строковые литералы — это способ создания строк в программе. Они представляют собой набор символов, заключенных в двойные кавычки или одинарные кавычки. Например:

```python
строка = "Привет, мир!"
```

В этом примере мы создали строковый литерал с помощью двойных кавычек.

### Индексация

Индексация — это способ доступа к конкретному символу в строке. Это делается с помощью квадратных скобок и индекса, который представляет собой целое число от 0 до длины строки минус 1. Например:

```python
строка = "Привет, мир!"
символ = строка[0]
```

В этом примере мы доступились к первому символу строки с помощью индекса 0.

### Срезы

Срезы — это способ получения части строки. Это делается с помощью квадратных скобок и двух индексов: начального и конечного. Например:

```python
строка = "Привет, мир!"
срез = строка[1:5]
```

В этом примере мы получили часть строки от второго символа до пятого.

### Методы строк

Методы строк — это функции, которые можно вызывать на строках. Они предоставляют различные возможности для работы со строками. Например:

```python
строка = "Привет, мир!"
длина_строки = len(строка)
```

В этом примере мы использовали метод `len()` для получения длины строки.

### Конкатенация

Конкатенация — это способ соединения двух или более строк в одну. Это делается с помощью оператора `+`. Например:

```python
строка1 = "Привет"
строка2 = ", мир!"
конкатенированная_строка = строка1 + строка2
```

В этом примере мы соединили две строки в одну.

### Экранированные последовательности

Экранированные последовательности — это способ представления специальных символов, которые имеют особое значение в строках. Они начинаются с обратной косой черты (`\`) и представляют собой один из следующих символов:

- `\n` — новая строка
- `\t` — горизонтальная табуляция
- `\\` — обратная косая черта

Например:

```python
строка = "Привет\nМир!"
```

В этом примере мы использовали экранированную последовательность `\n` для представления новой строки.

### Строки байтов

Строки байтов представляют собой последовательность байт, которые могут содержать любые данные, включая символы, числа и управляющие последовательности. Они используются для хранения и передачи двоичных данных, таких как изображения или видео.

Например:
```python
byte_string = b'Hello, World!'
print(byte_string)  # б'Hello, World!'
```
В этом примере мы создали строку байтов `b'Hello, World!'`, которая содержит последовательность байт, представляющих текст «Hello, World!». Символ `b` перед строкой указывает на то, что это строка байтов.

### Строки Юникода

Строки Юникода представляют собой последовательность Unicode-кодов, которые могут содержать любые символы из Unicode-полного набора. Они используются для хранения и передачи текстовых данных, включая национальные символы и специальные символы.

Например:
```python
unicode_string = 'Hello, World!'
print(unicode_string)  # Hello, World!
```
В этом примере мы создали строку Юникода `'Hello, World!'`, которая содержит последовательность Unicode-кодов, представляющих текст «Hello, World!».

### Неформатированные строки

Неформатированная строка — это строка, которая не имеет определенного формата или структуры. Она может содержать любые данные, включая символы, числа и управляющие последовательности.

Например:
```python
unformatted_string = 'Hello, World! 123'
print(unformatted_string)  # Hello, World! 123
```
В этом примере мы создали неформатированную строку `'Hello, World! 123'`, которая содержит текст и число.

### Тройные кавычки

Тройные кавычки — это способ обозначения строки в Python. Они используются для создания строк с любыми данными, включая символы, числа и управляющие последовательности.

Например:
```python
triple_quotes_string = '''Hello,
World!'''
print(triple_quotes_string)  # Hello,\nWorld!
```
В этом примере мы создали строку с тройными кавычками `'''Hello,\nWorld!''`, которая содержит текст и управляющую последовательность `\n`.

### Многострочные блоки текста

Многострочный блок текста — это способ обозначения строки в Python, которая может содержать несколько строк. Они используются для создания длинных строк с разными данными.

Например:
```python
multi_line_string = """Hello,
World!
This is a multi-line string."""
print(multi_line_string)  # Hello,\nWorld!\nThis is a multi-line string.
```
В этом примере мы создали многострочный блок текста `"""Hello,\nWorld!\nThis is a multi-line string."""`, который содержит несколько строк.

### Литералы строк

Литерал строки — это способ обозначения строки в Python, которая может содержать любые данные, включая символы, числа и управляющие последовательности. Они используются для создания строк с разными данными.

Например:
```python
literal_string = 'Hello, World! 123'
print(literal_string)  # Hello, World! 123
```
В этом примере мы создали литерал строки `'Hello, World! 123'`, который содержит текст и число.

### Служебные символы

Служебные символы — это набор специальных символов, которые используются в языке программирования для управления потоком выполнения кода. Они не являются частью строки и не могут быть использованы как обычный текст.

Например, в Python служебными символами являются:

* `\n` - переход на новую строку
* `\t` - табуляция
* `\\` - обратная косая черта (используется для экранирования других символов)
* `\"` - двойная кавычка (используется для обозначения строки внутри другой строки)

Служебные символы используются для форматирования строк и управления потоком выполнения кода.

### Идентификаторы Юникода

Идентификаторы Юникода — это специальные символы, которые представляют собой уникальные идентификаторы для каждого символа в Юникоде. Они используются для обозначения конкретных символов в строках.

Например, идентификатор Юникода для буквы "а" — `\u0041`. Идентификатор Юникода для цифры 4 — `\u0034`.

Идентификаторы Юникода используются для работы с символами, которые не могут быть представлены обычными ASCII-символами.

### Литеральные выражения

Литеральные выражения — это константы, которые представляют собой конкретные значения в программе. Они используются для присвоения переменным определенных значений или для использования этих значений в выражениях.

Например:

* `42` - литеральное выражение для числа 42
* `"Hello"` - литеральное выражение для строки "Hello"
* `True` - литеральное выражение для логического значения True

Литеральные выражения используются для присвоения переменным определенных значений или для использования этих значений в выражениях.

### Повторение

Повторение — это оператор, который позволяет повторять выполнение блока кода определенное количество раз. Он используется для реализации циклов и других повторяющихся задач.

Например:

* `for i in range(5): print(i)` - повторяет блок кода 5 раз
* `while True: print("Hello")` - повторяет блок кода бесконечно

Повторение используется для реализации циклов и других повторяющихся задач.

### Оператор in

Оператор `in` — это оператор, который проверяет, находится ли значение в определенной коллекции (например, строке или списке). Он используется для проверки принадлежности значения к определенному набору.

Например:

* `"a" in "hello"` - проверяет, находится ли буква "а" в строке "hello"
* `4 in [1, 2, 3, 4]` - проверяет, находится ли число 4 в списке `[1, 2, 3, 4]`

Оператор `in` используется для проверки принадлежности значения к определенному набору.

### Преобразования типов

Преобразования типов — это операторы, которые позволяют преобразовать значение одного типа в значение другого типа. Они используются для работы с различными типами данных.

Например:

* `int("42")` - преобразует строку "42" в число 42
* `str(42)` - преобразует число 42 в строку "42"
* `float("3.14")` - преобразует строку "3.14" в число 3,14

Преобразования типов используются для работы с различными типами данных.

### Операции среза

Операции среза — это набор методов, которые позволяют извлекать подмножества символов из строки. Эти операции используются для доступа к конкретным символам или группам символов в строке.

Например, если у нас есть строка `hello`, мы можем использовать операцию среза для извлечения первого символа следующим образом:

```python
s = "hello"
print(s[0])  # Выводит: h
```

В этом примере мы используем квадратные скобки для доступа к конкретному символу в строке. Индекс `0` указывает на первый символ строки.

Мы также можем использовать операцию среза для извлечения группы символов, указав диапазон индексов:

```python
s = "hello"
print(s[1:3])  # Выводит: el
```

В этом примере мы используем квадратные скобки с двумя аргументами: начальный и конечный индексы. Мы извлекаем символы с индексов `1` до `2`, что дает нам строку `"el"`.

### Выражения форматирования

Выражения форматирования — это способ форматировать строки, используя специальные последовательности символов для указания формата. Эти выражения используются для создания строк с определенным форматом, например с фиксированной шириной или десятичной точкой.

Например, мы можем использовать выражение форматирования для вывода числа с двумя десятичными знаками:

```python
x = 3.14159
print(f"{x:.2f}")  # Выводит: 3.14
```

В этом примере мы используем фигурные скобки `f` для указания, что строка должна быть форматированной. Затем мы указываем `.2f`, чтобы указать, что число должно иметь два десятичных знака.

Мы также можем использовать выражения форматирования для вывода строк с определенным форматом:

```python
name = "John"
age = 30
print(f"Имя: {name}, Возраст: {age}")  # Выводит: Имя: John, Возраст: 30
```

В этом примере мы используем фигурные скобки `f` для указания, что строка должна быть форматированной. Затем мы указываем `{name}` и `{age}`, чтобы указать, что эти значения должны быть вставлены в строку.

### Преобразование типов

Преобразование типов — это способ преобразовать значение одного типа в значение другого типа. Эти операции используются для преобразования значений из одной формы в другую форму, например от целого числа к строке.

Например, мы можем использовать функцию `str()` для преобразования целого числа в строку:

```python
x = 123
print(str(x))  # Выводит: 123
```

В этом примере мы используем функцию `str()`, чтобы преобразовать значение `x` из целого числа в строку.

Мы также можем использовать функцию `int()` для преобразования строки в целое число:

```python
s = "123"
print(int(s))  # Выводит: 123
```

В этом примере мы используем функцию `int()`, чтобы преобразовать значение `s` из строки в целое число.

### Разбиение строки на компоненты

Разбиение строки на компоненты — это способ разделить строку на отдельные части, используя определенные разделители. Эти операции используются для разделения строк на более мелкие составляющие.

Например, мы можем использовать функцию `split()` для разделения строки на слова:

```python
s = "hello world"
print(s.split())  # Выводит: ['hello', 'world']
```

В этом примере мы используем функцию `split()`, чтобы разделить строку на отдельные слова.

Мы также можем использовать функцию `split()` с аргументом, чтобы указать конкретный разделитель:

```python
s = "hello,world"
print(s.split(","))  # Выводит: ['hello', 'world']
```

В этом примере мы используем функцию `split()`, указывая `,` как разделитель.

### Оператор % для форматирования строк

Оператор `%` — это способ форматировать строки, используя специальные последовательности символов для указания формата. Эти операции используются для создания строк с определенным форматом, например с фиксированной шириной или десятичной точкой.

Например, мы можем использовать оператор `%` для вывода числа с двумя десятичными знаками:

```python
x = 3.14159
print("%f" % x)  # Выводит: 3.141590
```

В этом примере мы используем оператор `%`, указывая `f` как формат.

Мы также можем использовать оператор `%` для вывода строк с определенным форматом:

```python
name = "John"
age = 30
print("Имя: %s, Возраст: %d" % (name, age))  # Выводит: Имя: John, Возраст: 30
```

В этом примере мы используем оператор `%`, указывая `s` и `d` как форматы для строк и целых чисел соответственно.

### Спецификаторы формата

Спецификаторы формата — это набор специальных последовательностей символов, которые используются в выражениях форматирования или операторах `%` для указания формата. Эти спецификаторы используются для создания строк с определенным форматом.

Например, мы можем использовать спецификатор `f` для вывода числа с двумя десятичными знаками:

```python
x = 3.14159
print(f"{x:f}")  # Выводит: 3.14
```

В этом примере мы используем фигурные скобки `f`, указывая `{x:f}` как формат.

Мы также можем использовать спецификатор `d` для вывода целого числа:

```python
x = 123
print(f"{x:d}")  # Выводит: 123
```

В этом примере мы используем фигурные скобки `f`, указывая `{x:d}` как формат.

Спецификаторы формата могут быть использованы в выражениях форматирования или операторах `%` для создания строк с определенным форматом.

### Форматирование строк

Форматирование строк — это процесс преобразования строки в желаемый вид, чтобы она соответствовала определенным требованиям. Это может включать изменение регистра букв, добавление пробелов или других символов, а также форматирование чисел и даты.

Например, мы можем использовать функцию `upper()` для преобразования строки в верхний регистр:

```python
строка = "hello"
print(строка.upper())  # Вывод: HELLO
```

Мы также можем использовать функцию `format()` для форматирования строк. Например, мы можем добавить пробел между словами:

```python
строка = "hello world"
print("{}. {}".format("Hello", "World"))  # Вывод: Hello World
```

### Метод format

Метод `format()` — это метод строк, который позволяет форматировать строку. Он принимает аргументы и вставляет их в строку вместо специальных маркеров.

Например:

```python
строка = "Hello, {name}!"
print(строка.format(name="John"))  # Вывод: Hello, John!
```

Метод `format()` также поддерживает позиционные аргументы. Например:

```python
строка = "Hello, {}! My name is {}."
print(строка.format("John", "Alice"))  # Вывод: Hello, John! My name is Alice.
```

### Позиционные аргументы

Позиционные аргументы — это аргументы, которые передаются методу `format()` в определенной позиции. Они указываются в квадратных скобках `{}` и соответствуют порядку аргументов.

Например:

```python
строка = "Hello, {}! My name is {}."
print(строка.format("John", "Alice"))  # Вывод: Hello, John! My name is Alice.
```

### Именованные аргументы

Именованные аргументы — это аргументы, которые передаются методу `format()` с именем. Они указываются в круглых скобках `{name}` и соответствуют имени аргумента.

Например:

```python
строка = "Hello, {name}! My name is {other_name}."
print(строка.format(name="John", other_name="Alice"))  # Вывод: Hello, John! My name is Alice.
```

### Код типов значений

Код типов значений — это специальный синтаксис, который позволяет указывать тип значения в методе `format()`. Он используется для форматирования чисел и даты.

Например:

```python
строка = "The answer is {}."
print(строка.format(42))  # Вывод: The answer is 42.
```

Мы также можем использовать код типов значений для форматирования даты. Например:

```python
import datetime

дата = datetime.date.today()
строка = "Today's date is {}."
print(строка.format(дата))  # Вывод: Today's date is YYYY-MM-DD.
```

### Неизменяемые типы

Неизменяемые типы — это типы данных, которые не могут быть изменены после их создания. Например, строки и числа являются неизменяемыми типами.

Например:

```python
строка = "hello"
строка[0] = "H"  # TypeError: 'str' object does not support item assignment
```

Мы также можем использовать функцию `tuple()` для создания неизменяемого кортежа. Например:

```python
кортеж = (1, 2, 3)
кортеж[0] = 10  # TypeError: 'tuple' object does not support item assignment
```

### Изменяемые типы

Изменяемыми типами называются те, которые могут быть изменены после их создания. Это означает, что можно добавлять или удалять элементы в коллекции, а также изменять значения переменных.

Например, список (list) является изменяемым типом. Если вы создадите список и добавите к нему новый элемент, он будет изменен:

```python
my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # [1, 2, 3, 4]
```

Аналогично, если вы создадите переменную и присвоите ей значение, а затем измените это значение, оно будет изменено:

```python
x = 5
x = 10
print(x)  # 10
```

### Неизменяемость строк

Строки в Python являются неизменяемыми типами. Это означает, что после создания строки ее нельзя изменить.

Например, если вы попытаетесь добавить новый элемент к строке, это не сработает:

```python
my_str = "hello"
my_str.append(" world")  # AttributeError: 'str' object has no attribute 'append'
```

Аналогично, если вы попытаетесь изменить значение переменной, которая хранит строку, это не сработает:

```python
x = "hello"
x[0] = "H"  # TypeError: 'str' object does not support item assignment
```

### Преобразование символов

Преобразованием символов называется процесс преобразования строки в другой тип данных, такой как целое число или булево значение.

Например, вы можете использовать функцию `ord()` для получения кода символа:

```python
my_str = "a"
print(ord(my_str))  # 97
```

Аналогично, вы можете использовать функцию `chr()` для получения символа по его коду:

```python
code = 97
print(chr(code))  # 'a'
```

Вы также можете использовать функцию `int()` для преобразования строки в целое число:

```python
my_str = "123"
print(int(my_str))  # 123
```

Аналогично, вы можете использовать функцию `float()` для преобразования строки в вещественное число:

```python
my_str = "123.45"
print(float(my_str))  # 123.45
```

Наконец, вы можете использовать функцию `bool()` для преобразования строки в булево значение:

```python
my_str = "True"
print(bool(my_str))  # True
```
