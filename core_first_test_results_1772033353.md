## Основные концепции

**Основные концепции: Работа со строками**

**1. Списки, строки, кортежи**

В языке Python существует три основных типа данных для хранения последовательностей элементов: списки (lists), строки (strings) и кортежи (tuples). 

- **Списки**: представляют собой изменяемые последовательности элементов, которые могут содержать повторяющиеся значения. Списки можно модифицировать после их создания.
- **Строки**: представляют собой неизменяемые последовательности символов, которые часто используются для хранения текста. Страницы также являются последовательностями элементов, но они не могут быть изменены после своего создания.
- **Кортежи**: представляют собой неизменяемые последовательности элементов, которые также могут содержать повторяющиеся значения. Кортежи не могут быть модифицированы после их создания.

**2. Файлы**

В языке Python существует функция для работы с файлами, которая позволяет читать и записывать данные в файлы на диске. 

- **Чтение из файла**: функция `open()` используется для открытия файла в режиме чтения (`'r'`).
- **Запись в файл**: функция `open()` также используется для открытия файла в режиме записи (`'w'`) или добавления (`'a'`).

**3. Типы данных**

В языке Python существует несколько типов данных, которые используются для хранения различных видов данных.

- **Целые числа (int)**: представляют собой целые числа с произвольной величиной.
- **Соответствующие числа (float)**: представляют собой десятичные числа с произвольной величиной.
- **Строки (str)**: представляют собой последовательности символов, которые часто используются для хранения текста.
- **Логические значения (bool)**: представляют собой логические значения `True` или `False`.

**4. Динамическая типизация**

В языке Python используется динамическая типизация, которая означает, что тип данных объекта определяется во время выполнения программы, а не во время компиляции.

- **Типизация на лету**: тип данных объекта может быть изменен в течение выполнения программы.
- **Нет необходимости указывать тип данных**: тип данных объекта можно определить автоматически по его значению.

**5. Интерпретатор Python**

Интерпретатор Python — это программа, которая выполняет код на языке Python.

- **Выполняет код на лету**: интерпретатор выполняет код на лету, а не компилирует его в машинный код.
- **Поддерживает динамическую типизацию**: интерпретатор поддерживает динамическую типизацию, что позволяет определять тип данных объекта во время выполнения программы.

**6. Инструкции присваивания**

Инструкции присваивания — это операторы, которые назначают значение переменной.

- **Присвоение значения**: инструкция `=` используется для присвоения значения переменной.
- **Присвоение значения с учетом типов данных**: интерпретатор Python автоматически определяет тип данных переменной в зависимости от присваиваемого значения.

**7. Выражения и print**

Выражения — это части кода, которые вычисляются и возвращают значение.

- **Простые выражения**: простые выражения представляют собой операции над константами или переменными.
- **Сложные выражения**: сложные выражения представляют собой комбинацию простых выражений с помощью операторов.

**8. Итерации и генераторы**

Итерации — это механизм, который позволяет повторно использовать значения в цикле.

- **Функция `for`**: функция `for` используется для создания итератора.
- **Генераторы**: генераторы представляют собой функции, которые возвращают значение по мере необходимости.

**9. Функции**

Функции — это блок кода, который можно вызывать повторно с разными аргументами.

**10. Модули**

- **Использование функций и переменных из модуля**: функции и переменные из модуля используются как обычные функции или переменные.
