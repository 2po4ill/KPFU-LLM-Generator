**Основные концепции**

В этом разделе мы рассмотрим основные концепции числовых типов в Python, включая базовые числовые типы, преобразование типов, перегрузку операторов и полиморфизм.

**Базовые числовые типы**

Python имеет два основных числовых типа: целые числа (int) и вещественные числа (float). Целые числа представляют собой неограниченные целые числа, которые могут быть положительными или отрицательными. Вещественные числа представляют собой десятичные числа с плавающей точкой.

**Преобразование типов**

В Python существует возможность принудительного преобразования типов с помощью встроенных функций. Например, можно преобразовать вещественное число в целое число с помощью функции int:
```
>>> int(3.1415) # Усекает дробную часть вещественного числа
3
```
Аналогично, можно преобразовать целое число в вещественное число с помощью функции float:
```
>>> float(3) # Преобразует целое число в вещественное
3.0
```
Однако в обычных ситуациях делать это не приходится, потому что Python автоматически выполняет преобразование типов и тип результата, как правило, соответствует ожиданиям.

**Перегрузка операторов**

В языке Python существует возможность перегружать операторы с помощью классов или расширений на языке C для работы с создаваемыми объектами. Например, можно перегрузить операцию сложения для своих классов:
```
class MyClass:
    def __add__(self, other):
        return self.value + other
```
Кроме того, Python сам автоматически перегружает некоторые операторы, чтобы с их помощью можно было выполнять различные действия в зависимости от типа встроенных объектов. Например, оператор `+` выполняет операцию сложения для чисел, но когда он применяется к последовательностям, такими как строки или списки, он выполняет операцию конкатенации.

**Полиморфизм**

Полиморфизм — это возможность выполнять одну и ту же операцию над объектами разных типов. В Python полиморфизм достигается путем перегрузки операторов или использования функций, которые могут работать с различными типами данных.

**Примеры**

Давайте рассмотрим несколько примеров использования числовых типов и их преобразования:
```
>>> a = 3 # Создается целочисленное значение
>>> b = 4.0 # Создается вещественное значение
>>> a + b # Сложение целого числа и вещественного числа
7.0
>>> int(b) # Преобразование вещественного числа в целое число
4
```
Аналогично, можно перегрузить операцию сложения для своих классов:
```
class MyClass:
    def __add__(self, other):
        return self.value + other

obj1 = MyClass(3)
obj2 = MyClass(4.0)
print(obj1 + obj2) # Выводит 7.0
```
**Операции сравнения**

В Python можно сравнивать числа с помощью операторов сравнения:
```
>>> 1 < 2 # Меньше чем
True
>>> 2.0 >= 1 # Больше или равно: число 1 преобразуется в 1.0

>>> 2.0 == 2.0 # Проверка на равенство значений

```
**Деление**

В Python существует три версии операции деления:

* Классическое деление (Python 2.6 и более ранние версии)
* Истинное деление (Python 3.0)
* Деление с округлением вниз (Python 2.2 и более поздние версии)

Оператор `/` в Python 3.0 выполняет истинное деление, а оператор `//` выполняет деление с округлением вниз.

**Примеры**

Давайте рассмотрим несколько примеров использования операций деления:
```

>>> 10 // 4 # Деление с округлением вниз

>>> 10 / 4.0 # Истинное деление

>>> 10 // 4.0 # Деление с округлением вниз
2.0
```
