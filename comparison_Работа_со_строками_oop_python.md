**Работа со строками**

Строки являются одной из основных структур данных в Python. Они представляют собой последовательность символов, заключенных в кавычки. Структура строки определяется ее содержанием, а не длиной или типом элементов.

В Python для работы со строками используются следующие методы:

* `len()`: возвращает длину строки
* `str()` : конвертирует значение в строку
* `int()` : конвертирует строку в целое число
* `float()` : конвертирует строку в вещественное число
* `split()` : разбивает строку на подстроки по разделителю
* `join()` : объединяет строки в одну строку
