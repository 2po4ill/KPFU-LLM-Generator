**Основная часть**

### Словари

Словарь - это коллекция ключей и значений, которые можно использовать для быстрого доступа к данным. [ТРЕБУЕТ ПРОВЕРКИ (уверенность: -38%): Ключи в словаре должны быть уникальными, а значения могут быть любыми типами данных.]

**Определение словаря**

[ТРЕБУЕТ ПРОВЕРКИ (уверенность: 40%): Словарь определяется с помощью фигурных скобок `{}` и ключей-значений, разделенных двоеточием `:`.] Например:
```python
person = {'name': 'John', 'age': 30}
```
В этом примере мы создали словарь `person` с двумя ключами: `'name'` и `'age'`, а также соответствующими значениями.

**Доступ к элементам словаря**

[ТРЕБУЕТ ПРОВЕРКИ (уверенность: 7%): Чтобы получить значение по ключу, можно использовать квадратные скобки `[]`.] Например:
```python
print(person['name'])  # Выводит 'John'
```
[ТРЕБУЕТ ПРОВЕРКИ (уверенность: -22%): Если ключ не существует в словаре, будет выдано исключение `KeyError`.]

**Манипуляции с словарями**

Словари поддерживают следующие методы:

* `update()`: Обновляет словарь из другого словаря или коллекции.
* `get()`: Возвращает значение по ключу или None, если ключ не существует.
* `keys()`: Возвращает коллекцию ключей в словаре.
* `values()`: Возвращает коллекцию значений в словаре.

Например:
```python
person.update({'city': 'New York'})
print(person.get('city'))  # Выводит 'New York'
```
### Множества

[ТРЕБУЕТ ПРОВЕРКИ (уверенность: 34%): Множество - это набор уникальных элементов, который можно использовать для проверки принадлежности элемента к набору.]

**Определение множества**

[ТРЕБУЕТ ПРОВЕРКИ (уверенность: 49%): Множество определяется с помощью фигурных скобок `{}` и элементами, разделенными запятыми `,`.] Например:
```python
numbers = {1, 2, 3, 4, 5}
```
В этом примере мы создали множество `numbers` с пятью уникальными элементами.

**Операции над множествами**

Множества поддерживают следующие операции:

* `union()`: Возвращает объединение двух множеств.
* `intersection()`: Возвращает пересечение двух множеств.
* `difference()`: Возвращает разницу между двумя множествами.

Например:
```python
numbers1 = {1, 2, 3}
numbers2 = {3, 4, 5}

print(numbers1.union(numbers2))  # Выводит {1, 2, 3, 4, 5}
```
### Примеры использования словарей и множеств

Словари и множества имеют многочисленные применения в реальных проектах. Например:

* Словари можно использовать для хранения данных о пользователях или продуктах.
* Множества можно использовать для проверки принадлежности элемента к набору или для вычисления пересечения двух наборов.

Например:
```python
# Использование словаря для хранения данных о пользователе
user = {'name': 'John', 'age': 30, 'city': 'New York'}

# Использование множества для проверки принадлежности элемента к набору
numbers = {1, 2, 3}
print(4 in numbers)  # Выводит False
```

* Множества можно использовать для проверки принадлежности элемента к набору или для вычисления пересечения двух наборов.
