### 1. Числа

[ТРЕБУЕТ ПРОВЕРКИ (уверенность: -2%): Число — это значение, которое может быть использовано в вычислениях или логических операциях.] [ТРЕБУЕТ ПРОВЕРКИ (уверенность: -32%): В Python существует несколько типов чисел: целые числа, вещественные числа и комплексные числа.]

*   **Целые числа**: [ТРЕБУЕТ ПРОВЕРКИ (уверенность: -4%): Целое число — это число без десятичной части.] Например: 5, -3.
*   **Вещественные числа**: [ТРЕБУЕТ ПРОВЕРКИ (уверенность: 6%): Вещественное число — это число с десятичной частью.] Например: 3.14, -0.5.
*   **Комплексные числа**: [ТРЕБУЕТ ПРОВЕРКИ (уверенность: -32%): Комплексное число — это число со сложным действительным и мнимым членом.] Например: 2+3j, -1-4j.

### 2. Арифметические операции

Арифметические операции используются для выполнения различных математических функций, таких как сложение, вычитание, умножение и деление.

*   **Сложение**: [ТРЕБУЕТ ПРОВЕРКИ (уверенность: 2%): Сложение — это операция, которая добавляет два числа вместе.] Например: 2 + 3 = 5.
*   **Вычитание**: [ТРЕБУЕТ ПРОВЕРКИ (уверенность: -28%): Вычитание — это операция, которая вычитает одно число из другого.] Например: 5 - 3 = 2.
*   **Умножение**: [ТРЕБУЕТ ПРОВЕРКИ (уверенность: -27%): Умножение — это операция, которая умножает два числа вместе.] Например: 2 \* 3 = 6.
*   **Деление**: [ТРЕБУЕТ ПРОВЕРКИ (уверенность: 23%): Деление — это операция, которая делит одно число на другое.] Например: 6 / 2 = 3.

### 3. Сравнения

Сравнения используются для проверки равенства или неравенства двух чисел.

*   **Равно** (==): Равно — это операция, которая проверяет, являются ли два числа равными. Например: 2 == 2 = True.
*   **Не равно** (!=): Не равно — это операция, которая проверяет, не являются ли два числа равными. Например: 2 != 3 = True.
*   **Больше** (>): Больше — это операция, которая проверяет, является ли одно число больше другого. Например: 5 > 3 = True.
*   **Меньше** (<): Меньше — это операция, которая проверяет, является ли одно число меньше другого. Например: 3 < 5 = True.
*   **Больше или равно** (>=): Больше или равно — это операция, которая проверяет, являются ли два числа равными или одно из них больше другого. Например: 5 >= 3 = True.
*   **Меньше или равно** (<=): Меньше или равно — это операция, которая проверяет, являются ли два числа равными или одно из них меньше другого. Например: 3 <= 5 = True.

### 4. Логические операции

Логические операции используются для выполнения логических функций, таких как и, или, не.

*   **И** (and): И — это операция, которая проверяет, являются ли оба условия истинными. Например: a > 5 and b < 3 = True.
*   **Или** (or): Или — это операция, которая проверяет, является ли хотя бы одно из условий истинным. Например: a > 5 or b < 3 = True.
*   **Не** (not): Не — это операция, которая проверяет, является ли условие ложным. Например: not (a > 5) = False.

### Примеры

#### 1. Арифметические операции

```python
# Сложение
a = 2 + 3
print(a)  # Выводит 5

# Вычитание
b = 5 - 3
print(b)  # Выводит 2

# Умножение
c = 2 * 3
print(c)  # Выводит 6

# Деление
d = 6 / 2
print(d)  # Выводит 3.0
```

#### 2. Сравнения

```python
# Равно
a = 2
b = 2
if a == b:
    print("Числа равны")
else:
    print("Числа не равны")

# Не равно
c = 3
d = 5
if c != d:
    print("Числа не равны")
else:
    print("Числа равны")

# Больше
e = 5
f = 3
if e > f:
    print("Пятое число больше третьего")
else:
    print("Пятое число не больше третьего")

# Меньше
g = 3
h = 5
if g < h:
    print("Третье число меньше пятого")
else:
    print("Третье число не меньше пятого")

# Больше или равно
i = 5
j = 3
if i >= j:
    print("Пятое число больше или равно третьему")
else:
    print("Пятое число не больше или равно третьему")

# Меньше или равно
k = 3
l = 5
if k <= l:
    print("Третье число меньше или равно пятому")
else:
    print("Третье число не меньше или равно пятому")
```

#### 3. Логические операции

```python
# И
a = 2
b = 5
if a > 1 and b < 10:
    print("Условие истинно")

# Или
c = 3
d = 2
if c > 1 or d < 10:
    print("Условие истинно")

# Не
e = 5
f = 3
if not (e > f):
    print("Условие ложно")
```
