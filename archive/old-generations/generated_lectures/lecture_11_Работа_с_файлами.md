**Работа с файлами**

**Введение**

Работа с файлами - это фундаментальная задача в программировании, которая позволяет хранить и обрабатывать данные на диске. В этом разделе мы рассмотрим основы работы с файлами в Python.

Актуальность темы заключается в том, что работа с файлами является необходимостью в большинстве приложений, где требуется сохранение или загрузка данных. Например, если вы создаете программу для хранения контактов, вам нужно будет работать с файлом, чтобы сохранить и считать данные.

Цели и задачи лекции:

* Ознакомить студентов с основами работы с файлами в Python.
* Показать, как открывать и закрывать файлы.
* Рассмотреть различные режимы открытия файлов (чтение, запись, добавление).
* Привести примеры кода для демонстрации работы с файлами.

**Основная часть**

В Python работа с файлами начинается с использования функции `open()`. Эта функция открывает файл в заданном режиме и возвращает объект-файл, который можно использовать для чтения или записи данных.

Режимы открытия файла:

* `r` - режим чтения (по умолчанию).
* `w` - режим записи.
* `a` - режим добавления (добавляет данные в конец файла).

Например, чтобы открыть файл для записи, мы можем использовать следующий код:
```python
файл = open('example.txt', 'w')
```
Затем мы можем записать данные в файл с помощью метода `write()`:
```python
файл.write('Привет, мир!')
файл.close()
```
Если мы хотим открыть тот же файл для чтения, нам не нужно указывать режим:
```python
файл = open('example.txt', 'r')
данные = файл.read()
файл.close()
print(данные)  # Выводит: Привет, мир!
```
Мы также можем использовать метод `readline()` для чтения файла построчно:
```python
файл = open('example.txt', 'r')
while True:
    строка = файл.readline()
    if не строка:
        break
    print(строка)
файл.close()
```
**Примеры**

1. **Открытие и запись файла**
```python
файл = open('example.txt', 'w')
файл.write('Привет, мир!')
файл.close()
```
2. **Чтение файла**
```python
файл = open('example.txt', 'r')
данные = файл.read()
файл.close()
print(данные)  # Выводит: Привет, мир!
```
3. **Добавление данных в конец файла**
```python
файл = open('example.txt', 'a')
файл.write('\nЭто добавлено в конец файла.')
файл.close()
```
4. **Чтение файла построчно**
```python
файл = open('example.txt', 'r')
while True:
    строка = файл.readline()
    if не строка:
        break
    print(строка)
файл.close()
```
5. **Работа с файлами в цикле**
```python
файлы = ['file1.txt', 'file2.txt', 'file3.txt']
для файла в файлах:
    try:
        файл = open(файл, 'r')
        данные = файл.read()
        print(f'Файл: {файл}, Данные: {данные}')
        файл.close()
    except FileNotFoundError:
        print(f'Файл {файл} не найден.')
```
**Практические рекомендации**

* Всегда закрывайте файлы после использования.
* Используйте режим `with` для открытия и закрытия файлов автоматически.
* Для чтения больших файлов используйте метод `readline()` или `readlines()`.
* Для записи данных в файл используйте метод `write()`.
