Определение функций:

Функция определяется при помощи зарезервированного слова `def`. После этого слова указывается имя функции, за которым следует пара скобок, в которых можно указать имена некоторых переменных, и заключительное двоеточие в конце строки. Далее следует блок команд, составляющий функцию.

Например:
```python
def sayHello():
    print('Привет, Мир!')
```
В этом примере мы определяем функцию `sayHello`, которая принимает никаких аргументов и содержит только один блок команд - печать строки 'Привет, Мир!'.

Аргументы функций:

Функции могут принимать аргументы, которые передаются в функцию при её вызове. [ТРЕБУЕТ ПРОВЕРКИ (уверенность: 46%): Аргументы можно указывать в скобках после имени функции.]

Например:
```python
def sayHello(name):
    print('Привет, ' + name + '!')

sayHello('Мир')
```
В этом примере мы определяем функцию `sayHello`, которая принимает аргумент `name`. При вызове функции мы передаём значение `'Мир'` в качестве аргумента.

Возвращаемые значения функций:

[ТРЕБУЕТ ПРОВЕРКИ (уверенность: 11%): Функции могут возвращать значения, которые можно использовать в программе.] [ТРЕБУЕТ ПРОВЕРКИ (уверенность: -1%): Возвращаемое значение функции указывается с помощью оператора `return`.]

Например:
```python
def getSum(a, b):
    return a + b

result = getSum(2, 3)
print(result)  # выведет 5
```
В этом примере мы определяем функцию `getSum`, которая принимает два аргумента и возвращает их сумму. При вызове функции мы сохраняем результат в переменной `result` и выводим его на экран.

Исключения в функциях:

Функции могут выбрасывать исключения, которые можно обработать в программе. Исключения указываются с помощью оператора `raise`.

Например:
```python
def divide(a, b):
    if b == 0:
        raise ValueError('Деление на ноль!')
    return a / b

try:
    result = divide(2, 0)
except ValueError as e:
    print(e)  # выведет 'Деление на ноль!'
```
В этом примере мы определяем функцию `divide`, которая принимает два аргумента и возвращает их частное. При вызове функции мы проверяем, не равно ли второй аргумент нулю, и если да, то выбрасываем исключение `ValueError` с соответствующей ошибкой.
