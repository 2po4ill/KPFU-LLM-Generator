**Основные концепции**

В Python для работы с файлами используется модуль `os` и функции `open()` и `close()`. Файл можно открыть в режиме чтения (`'r'`) или записи (`'w'`). Если файл не существует, он будет создан. Если файл уже существует, его содержимое будет перезаписано.

Функция `open()` возвращает объект файла, который можно использовать для считывания или записи данных. Функция `close()` используется для закрытия файла после работы с ним.

```python
# Пример со страницы 115
class SchoolMember:
    def __init__(self, name, age):
        self.name = name
        self.age = age
        print('(Создан SchoolMember: {0})'.format(self.name))
    def tell(self):
        print('Имя:"{0}" Возраст:"{1}"'.format(self.name, self.age), end=" ")
```

```python
# Пример со страницы 116
class Teacher(SchoolMember):
    def __init__(self, name, age, salary):
        SchoolMember.__init__(self, name, age)
        self.salary = salary
        print('(Создан Teacher: {0})'.format(self.name))
    def tell(self):
        SchoolMember.tell(self)
        print('Зарплата: "{0:d}"'.format(self.salary))
```
