**Словари и множества**

В этой лекции мы обсудим два важных понятия в Python: словари (дictionaries) и множества (sets). Мы также рассмотрим пакеты (packages), которые являются удобным способом организовать модули.

**Основные концепции**

* **Словарь**: Словарь — это структура данных, которая хранит пары ключ-значение. Ключи должны быть уникальными, а значения могут повторяться.
* **Множество**: Множество — это структура данных, которая содержит неповторяющиеся элементы. В множестве можно хранить любые типы данных.
* **Пакет**: Пакет — это каталог, который содержит модули и специальный файл __init__.py, который указывает Python, что этот каталог является пакетом.

**Примеры кода**

```python
# Пример со страницы 83
shoplist = ['яблоки', 'манго', 'морковь', 'бананы']
print('Я должен сделать ', len(shoplist), 'покупок.')
```

```python
# Пример со страницы 84
mydict = {'name': 'John', 'age': 30}
print(mydict['name'])
```

```python
# Пример со страницы 85
myset = {1, 2, 3, 4, 5}
print(len(myset))
```

**Практические советы**

* Когда создаете словарь или множество, обязательно проверяйте, что ключи и элементы уникальны.
* Используйте пакеты для организовывать модули в удобной форме.
