**Списки и кортежи**

В этом разделе мы рассмотрим основные концепции списков и кортежей в языке Python.

**Введение**

Списки и кортежи - это два вида коллекций, которые можно использовать для хранения и манипулирования данными. Списки являются изменяемыми, то есть их содержимое можно модифицировать после создания, в отличие от кортежей, которые неизменяемы.

**Основные концепции**

Список - это изменяемая коллекция элементов, которые могут быть любого типа данных. Списки создаются с помощью квадратных скобок `[]` и могут содержать как простые значения, так и другие списки или кортежи.

Кортеж - это неизменяемая коллекция элементов, которая может содержать только простые значения или кортежи. Кортежи создаются с помощью круглых скобок `()`.

**Примеры кода**

```python
# Пример со страницы 71
import sys
print('Аргументы командной строки:')
for i in sys.argv:
    print(i)
```
