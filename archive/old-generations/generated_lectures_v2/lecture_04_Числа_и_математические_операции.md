**Основные концепции**

В Python существует два основных типа данных: числа и строки. Числа представляют собой целые или дробные числа, а строки — это последовательности символов, окруженных кавычками.

* **Числа**: В Python можно использовать десятичную точку для представления дробных чисел. Например: `3.14`.
* **Строки**: В Python строки окружаются кавычками и могут содержать любые символы, включая пробелы и табуляции. Например: `'Hello World'`.

**Примеры кода**

```python
# Пример со страницы 36
print('hello world')
```

```python
# Пример со страницы 38
print('Привет, Мир!')
```

```python
# Пример со страницы 49
i = 5
print(i)
i = i + 1
print(i)
s = '''Это многострочная строка.
Это вторая её строчка.'''
print(s)
```

**Практические советы**

* Используйте одинарные кавычки для представления строк, а двойные кавычки — для представления строк с пробелами или табуляциями.
* Используйте метод `format()` для форматирования строк и представления данных в определенной форме.
