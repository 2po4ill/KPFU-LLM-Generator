**Основы ООП: Классы, объекты, методы, наследование**

В этой лекции мы рассмотрим основные понятия объектно-ориентированного программирования (ООП), которые включают в себя классы, объекты, методы и наследование.

**1. Введение**

Объектно-ориентированное программирование — это парадигма программирования, которая основана на представлении данных как объектов и операций над этими объектами. [ТРЕБУЕТ ПРОВЕРКИ (уверенность: 12%): Классы и объекты являются фундаментальными понятиями ООП.]

**2. Классы**

Класс — этоblueprint или шаблон для создания объектов. Он определяет свойства и поведение объекта. [ТРЕБУЕТ ПРОВЕРКИ (уверенность: 13%): В Python класс создаётся ключевым словом `class`.]

**3. Объекты**

Объект — это экземпляр класса, который имеет собственные значения для его свойств (полей) и методов (функций). Объекты создаются на основе классов.

**4. Методы**

Методы — это функции, которые относятся к объекту или классу. Они определяют поведение объекта или класса. [ТРЕБУЕТ ПРОВЕРКИ (уверенность: 27%): В Python методы записываются в блоке класса и вызываются на объекте.]

**5. Переменные класса и объекта**

Данные, т.е. поля, являются частью классов и объектов. Поля разделяются между классом и объектом. Переменные класса доступны для всех экземпляров класса, а переменные объекта принадлежат конкретному объекту.

**6. Наследование**

[ТРЕБУЕТ ПРОВЕРКИ (уверенность: 9%): Наследование — это механизм, который позволяет одному классу наследовать свойства и поведение от другого класса.] [ТРЕБУЕТ ПРОВЕРКИ (уверенность: 19%): В Python наследование достигается с помощью ключевого слова `class`.]

**Примеры кода**

Ниже приведены примеры кода, которые демонстрируют основные понятия ООП:

```python
# Пример создания класса
class Person:
    def __init__(self, name):
        self.name = name

    def sayHi(self):
        print(f"Hello, {self.name}!")

# Создание объекта на основе класса
person = Person("John")
person.sayHi()  # Выводит: Hello, John!
```

```python
# Пример наследования
class Animal:
    def sound(self):
        print("Generic animal sound")

class Dog(Animal):
    def sound(self):
        print("Woof!")

dog = Dog()
dog.sound()  # Выводит: Woof!
```

**Практические рекомендации**

* Используйте ключевое слово `class` для создания классов.
* Используйте методы для определения поведения объекта или класса.
* Используйте наследование для создания новых классов на основе существующих.
