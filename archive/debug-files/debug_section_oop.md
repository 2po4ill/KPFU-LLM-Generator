# Введение

**Введение**

Объектно-ориентированное программирование (ООП) — это парадигма программирования, которая позволяет организовать программу как систему объектов, взаимодействующих между собой и решающих задачи. В этом разделе мы рассмотрим основы ООП и его ключевые понятия.

**Важность темы**

ООП является важнейшим направлением в разработке программного обеспечения, поскольку оно позволяет создавать гибкие, модульные и легко поддерживаемые системы. Оно также позволяет решить проблемы повторного использования кода, что является одной из главных проблем в разработке программного обеспечения.

**Применение**

ООП применяется во многих областях, включая бизнес-анализ, базы данных, компьютерное зрение и многое другое. Оно также используется в различных языках программирования, таких как Python, Java, C++ и многие другие.

**Классы и объекты**

В ООП класс — это сложный тип данных, включающий набор переменных (атрибутов) и функций (методов), которые управляют значениями этих переменных. Объект — это экземпляр класса, который имеет свои собственные атрибуты и методы.

**Например со страницы 17**

class MyClass:
    x = 10
    def __init__(self):
        self.y = 20

c1 = MyClass()
print(c1.x) # Вывод: 10
print(c1.y) # Вывод: 20

**Наследование**

Наследование — это ключевое понятие ООП, которое позволяет создавать новые классы на основе уже существующих классов. Дочерний класс наследует атрибуты и методы родительского класса и может дополнять или модифицировать их.

**Например со страницы 18**

class Animal:
    def __init__(self):
        self.name = "Animal"

class Dog(Animal):
    def __init__(self):
        super().__init__()
        self.breed = "Golden Retriever"

my_dog = Dog()
print(my_dog.name) # Вывод: Animal
print(my_dog.breed) # Вывод: Golden Retriever

**Инкапсуляция**

Инкапсуляция — это концепция ООП, которая заключается в объединении описания свойств объектов и их поведения в единое целое. Это позволяет скрыть внутреннюю реализацию объекта от внешнего мира.

**Например со страницы 19**

class BankAccount:
    def __init__(self):
        self.balance = 0

    def deposit(self, amount):
        self.balance += amount

my_account = BankAccount()
my_account.deposit(100)
print(my_account.balance) # Вывод: 100

**Полиморфизм**

Полиморфизм — это концепция ООП, которая заключается в возможности использования одного и того же имени для разных объектов. Это позволяет создавать гибкие системы, которые могут работать с разными типами данных.

**Например со страницы 20**

class Shape:
    def area(self):
        pass

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14 * (self.radius ** 2)

my_circle = Circle(5)
print(my_circle.area()) # Вывод: 78.54
