**9.2 Оператор while**

Оператор while позволяет многократно выполнять блок команд до тех пор, пока не будет выполнено некоторое условие. Это один из так называемых операторов цикла. Он также может иметь необязательный пункт else.

Пример: # Пример со страницы [62]
```python
number = 23
running = True
while running:
    guess = int(input('Введите целое число : '))
    if guess == number:
        print('Поздравляю, вы угадали.')
        running = False # это останавливает цикл while
    elif guess < number:
        print('Нет, загаданное число немного больше этого')
    else:
        print('Нет, загаданное число немного меньше этого.')
else:
    print('Цикл while закончен.')
# Здесь можете выполнить всё что вам ещё нужно
print('Завершение.')
```
В этом примере мы продолжаем игру в угадывание до тех пор, пока пользователь не угадает правильное число. Мы используем оператор while для многократного выполнения блока команд.

**9.3 Цикл for**

Оператор for..in также является оператором цикла, который осуществляет итерацию по последовательности объектов, т.е. проходит через каждый элемент в последовательности. Мы узнаем больше о последовательностях в дальнейших главах.

Пример: # Пример со страницы [63]
```python
for i in range(1, 5):
    print(i)
else:
    print('Цикл for закончен')
```
В этом примере мы выводим на экран последовательность чисел. Мы генерируем эту последовательность с помощью встроенной функции range.

**9.4 Оператор break**

Оператор break служит для прерывания цикла, т.е. остановки выполнения команд даже если условие выполнения цикла еще не приняло значение False или последовательность элементов не закончилась.

Пример: # Пример со страницы [65]
```python
while True:
    s = input('Введите что-нибудь : ')
    if s == 'выход':
        break
    print('Длина строки: ', len(s))
print('Завершение')
```
В этом примере мы многократно считываем пользовательский ввод и выводим на экран длину каждой строки. Для остановки программы мы вводим специальное условие, проверяющее, совпадает ли пользовательский ввод со строкой 'выход'.

**9.5 Оператор continue**

Оператор continue используется для указания Python, что необходимо пропустить все оставшиеся команды в текущем блоке цикла и продолжить с следующей итерацией цикла.

        print('Слишком мало')
        continue
    print('Введённая строка достаточной длины')

В этом примере мы запрашиваем пользовательский ввод, но обрабатываем его только если он имеет длину хотя бы в 3 символа. Если длина меньше 3, мы пропускаем остальные команды в блоке цикла и продолжаем с следующей итерацией.
