**7.4.6 Метод format() для форматирования строк**

Метод `format()` — это мощный инструмент для форматирования строк в Python. Он позволяет включать определенные обозначения в строку и затем вызывать метод `format()` для замены этих обозначений соответствующими аргументами.

Например, мы можем сохранить следующий текст в файле `str_format.py`:

```python
age = 26
name = 'Swaroop'
print('Возраст {0} -- {1} лет.'.format(name, age))
print('Почему {0} забавляется с этим Python?'.format(name))
```

Результатом выполнения этого кода будет:

```
$ python str_format.py
Возраст Swaroop -- 26 лет.
Почему Swaroop забавляется с этим Python?
```

Метод `format()` работает следующим образом: в строку могут быть включены определенные обозначения, а затем метод `format()` вызывается для замены этих обозначений соответствующими аргументами. В первом случае обозначение `{0}` соответствует переменной `name`, являющейся первым аргументом метода `format()`. Аналогично, второе обозначение `{1}` соответствует переменной `age`, являющейся вторым аргументом метода `format()`.

Заметьте, что мы можем добиться того же результата и без использования метода `format()` с помощью объединения строк:

```python
print('Возраст' + name + ' -- ' + str(age) + ' лет.')
```

Однако этот способ форматирования не так удобен, как использование метода `format()`, поскольку требует явного преобразования в строку и может привести к ошибкам.

**7.6 Имена идентификаторов**

Идентификаторы — это имена, присвоенные чему-то для его обозначения. Правила названия идентификаторов (переменных) в Python следующие:

* Первым символом идентификатора должна быть буква из алфавита (символ ASCII в верхнем или нижнем регистре, а также символ Unicode), либо знак подчёркивания (`_`).

* Примеры допустимых имён: `i`, `__my_name`, `name_23`, `a1b2_c3`.
* Примеры недопустимых имён: `2things`, `здесь есть пробелы`, `my-name`.

**7.9 Параметры функций**

Функции могут принимать параметры — некоторые значения, передаваемые функции для того, чтобы она что-либо сделала с ними.

Например:

```python
def printMax(a, b):
    if a > b:
        print(a, 'максимально')
    elif a == b:
        print(a, 'равно', b)
    else:
        print(b, 'максимально')
printMax(3, 4) # прямая передача значений
x = 5
y = 7
printMax(x, y) # передача переменных в качестве аргументов
```

В этом примере мы определяем функцию `printMax` с двумя параметрами `a` и `b`. Мы находим наибольшее число с помощью простого оператора `if..else` и выводим его значение.

При первом вызове функции `printMax` мы напрямую передаём числа в качестве аргументов. Во втором случае мы вызываем функцию с переменными в качестве аргументов. `printMax(x, y)` назначает значение аргумента `x` параметру `a`, а значение аргумента `y` — параметру `b`. В обоих случаях функция `printMax` работает одинаково.

**10.2 Локальные переменные**

При объявлении переменных внутри определения функции они не связаны с другими переменными с таким же именем за пределами функции — т.е. имена переменных являются локальными в функциях.

Например:

```python
x = 50
def func(x):
    print('x равен', x)
    x = 2
    print('Замена локального x на', x)
func(x)
print('x по прежнему', x)
```

В этом примере мы определяем функцию `func` с параметром `x`. Мы назначаем значение `x` в первой строке функции значение параметра, объявленного в основном блоке. Далее мы назначаем `x` значение `2`. Имя `x` локально для нашей функции. Поэтому когда мы заменим значение `x` в функции, значение `x`, объявленное в основном блоке, останется незатронутым.

Последний вызов функции `print` выводит значение `x`, указанное в основном блоке, подтверждая таким образом, что оно не изменилось при локальном присваивании значения в вызванной функции.

**10.3 Зарезервированное слово «global»**

Чтобы присвоить некоторое значение переменной, определённой на высшем уровне программы (т.е. не в какой-либо области видимости, как то функции или классов), необходимо указать Python, что это имя переменной локально, а глобально (`global`). Сделаем это с помощью зарезервированного слова `global`.

Без применения зарезервированного слова `global` невозможно присвоить значение переменной, определённой за пределами функции.

Например:

```python
x = 50
def func():
    global x
    print('x равно', x)
    x = 2
    print('Заменяем глобальное значение x на', x)
func()
print('Значение x составляет', x)
```

В этом примере мы определяем функцию `func` с зарезервированным словом `global`. Мы назначаем значение `x` в первой строке функции значение переменной, определённой за пределами функции. Далее мы назначаем `x` значение `2`. Имя `x` глобально для нашей функции. Поэтому когда мы заменим значение `x` в функции, значение `x`, объявленное за пределами функции, изменится.

**10.4 Зарезервированное слово «nonlocal»**

Мы увидели, как получить доступ к переменным в локальной и глобальной области видимости. Есть ещё один тип области видимости, называемый «нелокальной» (`nonlocal`), который представляет собой нечто среднее между первыми двумя.

Нелокальные области видимости встречаются, когда вы определяете функцию внутри другой функции.

Поскольку в Python всё является выполнимым кодом, вы можете определять функции где угодно.

**Метод format**

Метод `format()` используется для форматирования строк. Он позволяет заменить обозначения в строке на фактические значения. Например: `print('Возраст {0} -- {1} лет.'.format(name, age))` - здесь `{0}` и `{1}` являются обозначениями, которые заменяются на фактические значения `name` и `age`.
