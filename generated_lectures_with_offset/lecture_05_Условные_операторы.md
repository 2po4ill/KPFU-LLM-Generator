### Оператор if

Оператор `if` используется для проверки условия: если оно верно, выполняется блок команд (называемый «if-блок»), иначе выполняется другой блок команд (называемый «else-блок»). Блок «else» является необязательным.

Пример:
```
number = 23
guess = int(input('Введите целое число : '))
if guess == number:
    print('Поздравляю, вы угадали,') # Здесь начинается новый блок
    print('(хотя и не выиграли никакого приза!)') # Здесь заканчивается новый блок
elif guess < number:
    print('Нет, загаданное число немного больше этого.') # Ещё один блок
# Внутри блока вы можете выполнять всё, что угодно ...
else:
    print('Нет, загаданное число немного меньше этого.')
# чтобы попасть сюда, guess должно быть больше, чем number
print('Завершено')
# Это последнее выражение выполняется всегда после выполнения оператора if
```
Этот пример демонстрирует использование оператора `if` для проверки условия и выполнения соответствующего блока команд.

### Оператор elif

Оператор `elif` (сокращение от «else if») объединяет в себе два связанных `if-else` выражения в одно выражение `if-elif-else`. Это упрощает написание программы и сокращает количество необходимых отступов.

Пример:
```
# Пример со страницы 59
number = 23
guess = int(input('Введите целое число : '))
if guess == number:
    print('Поздравляю, вы угадали.')
elif guess < number:
    print('Нет, загаданное число немного больше этого')
else:
    print('Нет, загаданное число немного меньше этого.')
```
Этот пример демонстрирует использование оператора `elif` для проверки дополнительных условий.

### Оператор else

Оператор `else` является необязательным и выполняется только в том случае, если все предыдущие условия (`if` и `elif`) не верны.

Пример:
```
# Пример со страницы 59

    print('Поздравляю, вы угадали.')

    print('Нет, загаданное число немного больше этого')

    print('Нет, загаданное число немного меньше этого.')
```
Этот пример демонстрирует использование оператора `else` для выполнения блока команд в случае, когда все предыдущие условия не верны.

### Оператор while

Оператор `while` позволяет многократно выполнять блок команд до тех пор, пока определенное условие не станет ложным. Это один из типов операторов цикла.

Пример:
```
# Пример со странице 62

running = True
while running:
    guess = int(input('Введите целое число : '))
    if guess == number:
        print('Поздравляю, вы угадали.')
        running = False # это останавливает цикл while
    elif guess < number:
        print('Нет, загаданное число немного больше этого')
    else:
        print('Нет, загаданное число немного меньше этого.')
```
Этот пример демонстрирует использование оператора `while` для многократного выполнения блока команд до тех пор, пока определенное условие не станет ложным.

### Оператор for

Оператор `for` позволяет многократно выполнять блок команд для каждого элемента в последовательности. Это один из типов операторов цикла.

Пример:
```

Этот пример демонстрирует использование оператора `for` для многократного выполнения блока команд для каждого элемента в последовательности.

### Оператор break

Оператор `break` позволяет прервать цикл, т. е. остановить выполнение команд даже если условие цикла не стало ложным.

Этот пример демонстрирует использование оператора `break` для прерывания цикла.
