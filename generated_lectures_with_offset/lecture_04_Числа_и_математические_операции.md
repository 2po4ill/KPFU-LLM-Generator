**Основные концепции**

В этом разделе мы рассмотрим основные концепции Python, включая литеральные константы, комментарии, операторы и выражения.

**7.1 Комментарии**

Комментарии — это важнейший аспект программирования на Python. Они позволяют нам добавлять примечания к нашим кодам, которые могут быть полезны для других разработчиков или даже для самого себя в будущем. В Python комментарий начинается с символа # и продолжается до конца строки.

Например:
```python
print('Привет, Мир!) # print -- это функция
```
В этом примере мы добавили комментарий к строке `print('Привет, Мир!)`, который объясняет, что эта строка является вызовом функции `print`.

**7.2 Литеральные константы**

Литеральной константой называется значение, которое не изменяется в течение всей программы. В Python литеральные константы могут быть числами, строками или даже комплексными числами.

Например:
```python
5  # целое число
1.23  # число с плавающей точкой
9.25e-3  # комплексное число
```
В этом примере мы показали три литеральных константы: целое число `5`, число с плавающей точкой `1.23` и комплексное число `9.25e-3`.

**7.3 Числа**

Числами в Python являются три типа: целые, числа с плавающей точкой и комплексные.

* Целые числа — это числа без десятичной точки, такие как 2 или -5.
* Числа с плавающей точкой — это числа с десятичной точкой, такие как 3.14 или -0.25.
* Комплексные числа — это числа в форме a + bj, где a и b являются действительными числами.

Например:
```python
2  # целое число
3.14  # число с плавающей точкой
-5 + 3j  # комплексное число
```
В этом примере мы показали три типа чисел: целое число `2`, число с плавающей точкой `3.14` и комплексное число `-5 + 3j`.

**7.4 Строки**

Строками в Python являются последовательности символов, которые могут быть окружены одинарными или двойными кавычками.

Например:
```python
'Привет, Мир!'  # строка с одинарными кавычками
"What's your name?"  # строка с двойными кавычками
```
В этом примере мы показали две строки: одна с одинарными кавычками и другая с двойными кавычками.

**7.4.1 Одинарные кавычки**

Строки в Python можно указать, используя одинарные кавычки. В этом случае все пробелы и табуляции сохранятся как есть.

Например:
```python
'Привет, Мир!'  # строка с одинарными кавычками
```
В этом примере мы показали строку с одинарными кавычками. Обратите внимание, что пробелы и табуляции сохраняются как есть.

**7.4.2 Двойные кавычки**

Строки в Python можно указать, используя двойные кавычки. В этом случае все пробелы и табуляции сохранятся как есть.

Например:
```python
"What's your name?"  # строка с двойными кавычками
```
В этом примере мы показали строку с двойными кавычками. Обратите внимание, что пробелы и табуляции сохраняются как есть.

**7.4.3 Тройные кавычки**

Строки в Python можно указать, используя тройные кавычки. В этом случае внутри тройных кавычек можно использовать двойные или тройные кавычки.

Например:
```python
'''Это многострочная строка.
Это её первая строка.
"What's your name?", - спросил я.
Он ответил: "Bond, James Bond."
'''
```
В этом примере мы показали многострочную строку с тройными кавычками. Обратите внимание, что внутри тройных кавычек можно использовать двойные или тройные кавычки.

**7.4.4 Строки неизменяемы**

Строки в Python являются неизменяемыми объектами. Это означает, что после создания строки ее нельзя изменить.

Например:
```python
my_string = 'Привет, Мир!'
my_string[0] = 'З'
```
В этом примере мы пытаемся изменить первый символ строки `my_string`. Однако это невозможно, поскольку строки являются неизменяемыми объектами.

**7.4.5 Объединение строковых констант**

Если расположить рядом две строковые константы, Python автоматически их объединит.

Например:
```python
'What\'s ' 'your name?'  # автоматически объединяется в "What's your name?"
```
В этом примере мы показали, как две строковые константы объединяются в одну строку.

**8.1 Операторы**

Операторами в Python называются символы или слова, которые используются для выполнения операций над данными.

Например:
```python
2 + 3  # оператор сложения
```
В этом примере мы показали оператор сложения `+`.

**8.1. Операторы**

Операторы в Python можно разделить на несколько категорий:

* Арифметические операторы: `+`, `-`, `*`, `/`, `%`, `//`
* Сравнительные операторы: `<`, `>`, `==`, `!=`, `<=` , `>=`
* Логические операторы: `and`, `or`, `not`

Например:
```python
2 + 3  # арифметический оператор сложения
5 < 3  # сравнительный оператор меньше
True and False  # логический оператор И
```
В этом примере мы показали три оператора: арифметический оператор сложения `+`, сравнительный оператор меньше `<` и логический оператор И `and`.

**8.1.1 Краткая запись мат. операций и присваивания**

Python позволяет использовать краткую запись для некоторых математических операций и присваивания.

Например:
```python
a = 2; a *= 3  # краткая запись для умножения
```
В этом примере мы показали краткую запись для умножения `*=`.

**8.2 Порядок вычисления**

Порядок вычисления в Python определяется таблицей приоритета операторов.

Например:
```python
2 + 3 * 4  # порядок вычисления: сначала умножение, затем сложение
```
В этом примере мы показали порядок вычисления для выражения `2 + 3 * 4`.

**8.3 Изменение порядка вычисления**

Python позволяет использовать скобки для изменения порядка вычисления.

Например:
```python
(2 + 3) * 4  # измененный порядок вычисления: сначала сложение, затем умножение
```
В этом примере мы показали измененный порядок вычисления для выражения `(2 + 3) * 4`.

**8.3. Изменение порядка вычисления**

Изменение порядка вычисления в Python может быть полезно для улучшения читаемости кода.

Например:
```python
result = (a + b) * c  # измененный порядок вычисления: сначала сложение, затем умножение
```
В этом примере мы показали измененный порядок вычисления для выражения `(a + b) * c`.
