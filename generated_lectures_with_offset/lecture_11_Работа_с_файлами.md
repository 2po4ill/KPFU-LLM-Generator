**Типы файлов**

Файлы можно классифицировать по типу содержимого: текстовые файлы и двоичные файлы. Текстовые файлы содержат данные в виде строк или символов, тогда как двоичные файлы представляют собой набор битов (0 или 1), которые могут быть интерпретированы как числа, изображения или другие типы данных.

**Текстовые файлы**

Текстовые файлы являются наиболее распространенным типом файлов и используются для хранения данных в виде строк или символов. Файлы с расширением `.txt` обычно являются текстовыми файлами. Для работы с текстовыми файлами Python предоставляет функции `open()` и `read()`, которые позволяют читать содержимое файла.

**Чтение из текстового файла**

Чтобы прочитать содержимое текстового файла, можно использовать функцию `open()` в режиме «чтения» (`'r'`). Функция `open()` возвращает объект файлового потока, который можно использовать для чтения данных. Например:
```
# Пример со страницы 37
file = open('example.txt', 'r')
data = file.read()
print(data)
file.close()
```
В этом примере мы открываем файл `example.txt` в режиме «чтения» и читаем его содержимое с помощью функции `read()`. Содержимое файла выводится на экран.

**Запись в текстовый файл**

В этом примере мы открываем файл `example.txt` в режиме «записи» и записываем данные с помощью функции `write()`.

### Чтение файла

### Работа с двоичным файлом

Работа с двоичными файлами является еще одной важной операцией при работе с файлами. Функция `open()` используется для открытия файла, а затем мы можем читать содержимое файла с помощью метода `read()`. # Пример со страницы 39

### Режимы открытия файла

Функция `open()` принимает два аргумента: имя файла и режим открытия. Режим открытия определяет, как будет использоваться файл. # Пример со страницы 40
```python
# Режим чтения ('r')
file = open('example.txt', 'r')

# Режим записи ('w')
file = open('example.txt', 'w')

# Режим дозаписи ('a')
file = open('example.txt', 'a')

# Режим чтения и записи ('r+')
file = open('example.txt', 'r+')

# Режим двоичного чтения ('rb')
file = open('example.bin', 'rb')

# Режим двоичного записи ('wb')
file = open('example.bin', 'wb')
```
В этом примере мы демонстрируем различные режимы открытия файла: режим чтения (`'r'`), режим записи (`'w'`), режим дозаписи (`'a'`), режим чтения и записи (`'r+'`), режим двоичного чтения (`'rb'`) и режим двоичного записи (`'wb'`).

### Исключения

При работе с файлами могут возникать исключения. Например, если файл не существует или доступ к нему запрещен. # Пример со страницы 42
```python
try:
    file = open('example.txt', 'r')
except FileNotFoundError:
    print("Файл не найден")
```
В этом примере мы демонстрируем исключение `FileNotFoundError`, которое возникает, если файл не существует.

### Блоки try-except

Блоки try-except используются для обработки исключений. # Пример со страницы 43
```python
try:
    file = open('example.txt', 'r')
except FileNotFoundError:
    print("Файл не найден")
else:
    print(file.read())
finally:
    file.close()
```
В этом примере мы демонстрируем блок try-except, который включает в себя три блока: блок `try`, блок `except` и блок `finally`. Блок `try` содержит код, который может вызвать исключение. Блок `except` содержит код, который будет выполнен, если исключение возникнет. Блок `finally` содержит код, который всегда будет выполняться, независимо от того, возникло ли исключение или нет.
