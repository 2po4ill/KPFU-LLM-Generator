**Основные концепции**

В этом разделе мы рассмотрим основные концепции, связанные с структурами данных в языке Python. Мы начнем с кортежей, затем перейдем к спискам и закончим словарями.

### Кортежи

Кортеж — это неизменяемый тип последовательности, который содержит элементы, разделенные запятыми. Кортежи используются для хранения набора значений, которые не должны изменяться в течение выполнения программы. В отличие от списков, кортежи не могут быть модифицированы после их создания.

Кортежи можно создавать с помощью круглых скобок и запятой между элементами. Например:

```python
# Пример со страницы 91
singleton = (2,)
```

В этом примере мы создали кортеж, содержащий только один элемент — число 2.

Кортежи можно использовать в качестве аргументов функций и возвращать их как результаты функций. Они также могут быть использованы в циклах for для перебора их элементов.

### Списки

Список — это изменяемый тип последовательности, который содержит элементы, разделенные запятыми. Списки используются для хранения набора значений, которые могут быть модифицированы в течение выполнения программы.

Списки можно создавать с помощью квадратных скобок и запятой между элементами. Например:

```python
# Пример со страницы 91
shoplist = ['яблоки', 'манго', 'морковь', 'бананы']
```

В этом примере мы создали список, содержащий четыре элемента — названия фруктов.

Списки можно использовать в качестве аргументов функций и возвращать их как результаты функций. Они также могут быть использованы в циклах for для перебора их элементов.

Методы списков:

*   `append()`: добавляет новый элемент в конец списка.
*   `extend()`: добавляет несколько элементов в конец списка.
*   `insert()`: вставляет новый элемент на определенную позицию в списке.
*   `remove()`: удаляет первый встреченный элемент, равный заданному значению.
*   `sort()`: сортирует список по умолчанию или по заданному ключу.

### Словари

Словарь — это изменяемый тип данных, который содержит пары ключ-значение. Ключи словаря должны быть уникальными и не могут быть изменены после их создания. Значения в словаре могут быть любыми объектами.

Словари можно создавать с помощью фигурных скобок и запятой между парами ключ-значение. Например:

```python
# Пример со страницы 92
ab = { 'Swaroop' : 'swaroop@swaroopch.com',
       'Larry' : 'larry@wall.org',
       'Matsumoto' : 'matz@ruby-lang.org',
       'Spammer' : 'spammer@hotmail.com'
     }
```

В этом примере мы создали словарь, содержащий четыре пары ключ-значение — названия людей и их адреса электронной почты.

Словари можно использовать в качестве аргументов функций и возвращать их как результаты функций. Они также могут быть использованы в циклах for для перебора их элементов.

Методы словарей:

*   `keys()`: возвращает список ключей словаря.
*   `values()`: возвращает список значений словаря.
*   `items()`: возвращает список пар ключ-значение словаря.
*   `update()`: добавляет новые пары ключ-значение в словарь.
*   `pop()`: удаляет пару ключ-значение из словаря и возвращает ее значение.

### Последовательности

Последовательность — это тип данных, который содержит элементы, разделенные запятыми. Последовательности могут быть кортежами, списками или строками.

Методы последовательностей:

*   `in`: проверяет, находится ли заданный элемент в последовательности.
*   `not in`: проверяет, не находится ли заданный элемент в последовательности.
*   `index()`: возвращает индекс первого встреченного элемента в последовательности.
*   `count()`: возвращает количество вхождений заданного элемента в последовательность.

### Множества

Множество — это неупорядоченный набор уникальных объектов. Множества используются для хранения набора значений, которые не должны повторяться.

Методы множеств:

*   `add()`: добавляет новый элемент в множество.
*   `remove()`: удаляет первый встреченный элемент из множества.
*   `issuperset()`: проверяет, является ли множество подмножеством другого множества.
*   `intersection()`: возвращает пересечение двух множеств.

### Ссылки

Когда мы создаем объект и присваиваем его переменной, переменная только ссылается на объект, а не представляет собой этот объект. Это называется привязкой имени к объекту.

Пример:

```python
# Пример со страницы 97
shoplist = ['яблоки', 'манго', 'морковь', 'бананы']
mylist = shoplist # mylist - лишь ещё одно имя, указывающее на тот же объект!
del shoplist[0] # Я сделал первую покупку, поэтому удаляю её из списка
print('shoplist:', shoplist)
print('mylist:', mylist)
```

В этом примере мы создали список shoplist и присвоили его переменной mylist. Затем мы удалили первый элемент из shoplist, но mylist все еще ссылается на тот же объект, поэтому он также удалит первый элемент.

Это означает, что если вы хотите сохранить исходный список, вы должны сделать копию списка при помощи полной вырезки:

```python
# Пример со страницы 97
mylist = shoplist[:] # создаём копию путём полной вырезки
del mylist[0] # удаляем первый элемент
print('shoplist:', shoplist)
print('mylist:', mylist)
```

В этом случае mylist будет содержать копию исходного списка, а не ссылаться на него.
