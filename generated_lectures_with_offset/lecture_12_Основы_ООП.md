**Введение**

Объектно-ориентированное программирование (ООП) - это парадигма программирования, которая позволяет организовать программы как набор взаимосвязанных объектов. В этом разделе мы рассмотрим основные понятия ООП и их применение в языке Python.

**Структуры данных**

В Python существуют четыре встроенных структуры данных: список, кортеж, словарь и множество. Списки - это изменяемые структуры данных, которые могут содержать любые типы объектов, включая числа, строки и другие списки.

Список - это структура данных, которая содержит упорядоченный набор элементов. Он легко представить как список покупок, в котором перечисляется, что нужно купить, с тем исключением, что в списке покупок каждый элемент обычно размещается на отдельной строке, тогда как в Python они разделяются запятыми.

Список элементов должен быть заключен в квадратные скобки, чтобы Python понял, что это список. Как только список создан, можно добавлять, удалять или искать элементы в нем. Поскольку элементы можно добавлять и удалять, мы говорим, что список - это изменяемый тип данных.

Кортежи - это неизменяемые структуры данных, которые могут содержать любые типы объектов, включая числа, строки и другие кортежи. Кортежи обозначаются указанием элементов, разделенных запятыми; по желанию их можно заключить в круглые скобки.

Кортежи обычно используются в тех случаях, когда оператор или пользовательская функция должна знать, что набор значений, т.е. кортеж значений, не изменится.

**Объектно-ориентированное программирование**

До сих пор наши программы состояли из функций, т. е. блоков выражений, которые манипулируют данными. Это называется процедурно-ориентированным стилем программирования. Существует и другой способ организации программ: объединять данные и функционал внутри некоего объекта. Это называется объектно-ориентированной парадигмой программирования.

В большинстве случаев можно ограничиться процедурным программированием, а если решить конкретную задачу, то может понадобиться переход к техникам объектно-ориентированного программирования.

Два основных аспекта объектно-ориентированного программирования - классы и объекты. Класс создает новый тип, а объекты являются экземплярами класса. Аналогично, когда мы говорим «переменные типа int», это означает, что переменные, которые хранят целочисленные значения, являются экземплярами (объектами) класса int.

Обратите внимание, что даже целые числа рассматриваются как объекты (класса int), в отличие от C++ и Java (до версии 1.5), где целые числа являются примитивами. См. help(int) для более детального описания этого класса.

Кортеж — это неизменяемый тип данных, который может содержать любые объекты. Он представляет собой упорядоченный набор элементов. Кортеж можно создать с помощью круглых скобок: `my_tuple = ('питон', 'слон', 'пингвин')`.

Поля — это переменные, которые принадлежат классу или объекту. Они используются для хранения данных. Поля можно доступить через точку: `my_list.field`.

Метод `__init__` — это специальный метод, который вызывается при создании нового экземпляра класса. Он используется для инициализации объекта.

Советы по использованию методов и полей класса в Python:

1. **Используйте методы для реализации функциональности**: Методы — это функции, которые принадлежат классу или объекту. Они используются для реализации функциональности, связанной с данным классом или объектом.
