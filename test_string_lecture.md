**Работа со строками: форматирование, методы строк, срезы**

В этой лекции мы будем изучать основные концепции работы со строками в языке Python. Мы рассмотрим форматирование строк, методы строк и срезы.

**Основные концепции**

Строка - это последовательность символов, заключенных в кавычки. В Python можно использовать одинарные или двойные кавычки для обозначения строки.

Форматирование строк позволяет создать строки с конкретным форматом. Например, мы можем использовать `%s` для замены значения переменной в строке.

Методы строк - это функции, которые можно вызывать на строках. Они позволяют выполнять различные операции с строками, такие как поиск и замена символов.

Срезы - это способ доступа к конкретным символам или подстрокам строки.

**Примеры кода**

```python
# Пример со страницы 8
name = "John"
age = 25
print("Мое имя %s, мне %d лет." % (name, age))
```

```python
# Пример со страницы 9
str1 = "Hello"
str2 = "World"
print(str1 + str2)
```

```python
# Пример со страницы 10
str = "Python is a great language."
print(str.upper())
```

```python
# Пример со страницы 11
str = "Hello, World!"
print(str[0])
```

**Практические советы**

* Используйте `%s` для замены значения переменной в строке.
* Используйте `+` для соединения двух строк.
* Используйте методы строки, такие как `upper()` и `lower()`, для изменения формата строки.
* Используйте срезы для доступа к конкретным символам или подстрокам строки.
