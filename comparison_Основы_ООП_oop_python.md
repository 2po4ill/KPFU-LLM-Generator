class MyClass:

**Пример 2: Наследование**

На странице [22] учебника говорится, что наследование является одним из основных понятий ООП. Давайте рассмотрим пример наследования:

```python
class Parent:
    def __init__(self):
        self.x = 10

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.y = 20

child = Child()
print(child.x)  # вывод: 10
print(child.y)  # вывод: 20
```

В этом примере мы создали класс `Parent` с атрибутом `x`, затем наследуем его в классе `Child`. В методе `__init__` класса `Child` мы вызываем метод `super().__init__()`, чтобы инициализировать атрибуты родительского класса.

```python
class WinDoor:

```

В этом примере мы создали класс `WinDoor` с атрибутом `square`, затем создали класс `Room`, который включает в себя экземпляры класса `WinDoor`. В методе `add_wd` класса `Room` мы добавляем объекты-компоненты в список.

**Пример 4: Статические методы**

В этом примере мы создали класс `A` с статическим методом `meth`. Затем мы вызвали этот метод через класс, а не через объект.

**Создание классов и объектов**

Класс описывается с помощью ключевого слова `class` по следующей схеме:
```python
class <Название класса>[(<Класс1>, ..., <Класс№>)]:
    """ Строка документирования """
    <Описание атрибутов и методов>
```
Инструкция создает новый объект и присваивает ссылку на него идентификатору, указанному после ключевого слова `class`. Это означает, что название класса должно полностью соответствовать правилам именования переменных.

Атрибуты класса можно разделить на две группы: атрибуты первого типа принадлежат самому классу, а не конкретному экземпляру. Доступ к таким атрибутам вне тела класса осуществляется через точечную нотацию через объект объявления класса.

Атрибуты второго типа принадлежат конкретному экземпляру и создаются уже во время существования объекта. Чтобы создать экземпляр класса используется синтаксис `<экземпляр класса> = <Название класса>([<Параметры>])`.

**Наследование**

Наследование является одним из основных понятий ООП. Предположим, у нас есть класс (например, `Class1`). При помощи наследования мы можем создать новый класс (`Class2`), в котором будет реализован доступ ко всем атрибутам и методам класса `Class1`.

Класс `Class2` наследует все атрибуты и методы класса `Class1`. Если имя метода в классе `Class2` совпадает с именем метода класса `Class1`, то будет использоваться метод из класса `Class2`.
