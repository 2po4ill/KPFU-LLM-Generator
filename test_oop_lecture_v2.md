**Введение**

Объектно-ориентированное программирование (ООП) - это парадигма программирования, которая позволяет создавать сложные системы из простых компонентов. В ООП данные и функции объединяются в единую сущность - класс. Классы могут наследовать поведение друг друга, что позволяет создавать более сложные системы.

**Основные концепции**

* **Класс**: Класс - это шаблон, который определяет свойства и поведение объекта.
* **Объект**: Объект - это экземпляр класса, который имеет свои собственные значения для свойств и может выполнять методы.
* **Метод**: Метод - это функция, которая принадлежит к классу или объекту и определяет поведение этого класса или объекта.
* **Наследование**: Наследование - это механизм, который позволяет одному классу наследовать свойства и поведение другого класса.

**Примеры кода**

```python
# Пример со страницы 110
class Person:
    pass # Пустой блок

p = Person()
print(p)
```

```python
# Пример со страницы 112
class Person:
    def __init__(self, name):
        self.name = name
    def sayHi(self):
        print('Привет! Меня зовут', self.name)

p = Person('Swaroop')
p.sayHi()
```

```python
# Пример со страницы 113
class Person:
    def __init__(self, name):
        self.name = name
    def sayHi(self):
        print('Привет! Меня зовут', self.name)

p1 = Person('Swaroop')
p2 = Person('Guido')

print(p1.name) # Swaroop
print(p2.name) # Guido

p1.sayHi() # Привет! Меня зовут Swaroop
p2.sayHi() # Привет! Меня зовут Guido
```

**Практические советы**

* Используйте классы для создания сложных систем из простых компонентов.
* Наследование позволяет создавать более сложные системы, наследуя поведение друг друга.
* Объекты имеют свои собственные значения для свойств и могут выполнять методы.

**Заключение**

Объектно-ориентированное программирование - это мощный инструмент для создания сложных систем. Классы, объекты, методы и наследование - все эти концепции позволяют создавать более сложные системы из простых компонентов.
