## Основные концепции

**Основные концепции: Тлеющий разряд**

### Литеральные константы

Литеральные константы - это постоянные значения, которые используются в программе. Они могут быть числами, строками или логическими значениями (true/false). Литеральные константы не изменяются во время выполнения программы и всегда имеют одинаковое значение.

Пример:
```python
x = 5; y = "Hello"; z = true;
```
В этом примере `5`, `"Hello"` и `true` - это литеральные константы. Они не меняются во время выполнения программы и всегда будут иметь одно и то же значение.

### Числа

Числа - это литеральные константы, которые представляют собой целые или дробные числа. В языках программирования часто используются различные типы чисел, такие как `int`, `float` или `double`.

Пример:
```python
x = 5; y = 3.14;
```
В этом примере `5` и `3.14` - это числа. `5` является целым числом (`int`), а `3.14` - дробным числом (`float`).

### Строки

Строки - это литеральные константы, которые представляют собой последовательность символов. В языках программирования часто используются двойные кавычки для обозначения строк.

Пример:
```python
x = "Hello"; y = 'World';
```
В этом примере `"Hello"` и `'World'` - это строки. `x` и `y` имеют одинаковое значение, но разный тип символов (`"Hello"` - двойные кавычки, а `'World'` - одинарные кавычки).

### Операторы

Операторы - это символы или ключевые слова, которые используются для выполнения операций над литералами. Например, оператор `+` используется для сложения двух чисел.

Пример:
```python
x = 5 + 3;
```
В этом примере `+` - это оператор сложения. Он выполняется над двумя числами (`5` и `3`) и возвращает результат (`8`).

### Выражения

Выражения - это комбинация литеральных констант, операторов и других выражений, которые используются для вычисления значения. Например, выражение `x + 3` оценивается как `5 + 3`, что возвращает результат (`8`).

Пример:
```python
x = 5; y = x + 3;
```
В этом примере `x + 3` - это выражение. Оно оценивается как `5 + 3`, что возвращает результат (`8`). Результат присваивается переменной `y`.

### Комментарии

Комментарии - это текст, который добавляется к программе для пояснения ее функциональности или для предоставления информации о том, как она работает. Комментарии не влияют на выполнение программы и могут быть удалены без изменения ее поведения.

Пример:
```python
# Это комментарий
x = 5; # Это еще один комментарий
```
В этом примере `# Это комментарий` и `# Это еще один комментарий` - это комментарии. Они не влияют на выполнение программы и могут быть удалены без изменения ее поведения.

### Операнды

Операнды - это входные данные, которые используются в операциях. В языках программирования операнды могут быть переменными, константами или выражениями.

Например, в выражении `a = b + c` аргументами являются `b` и `c`, которые являются операндами.

Операнды могут быть:

- Переменные: `x`, `y`
- Константы: `5`, `10`
- Выражения: `a + b`, `x * y`

### Ассоциативность

Ассоциативность - это свойство операций, которое определяет порядок выполнения операций при наличии нескольких операндов.

Например, в выражении `a + b + c` можно выполнять сложение следующими способами:

- `(a + b) + c`
- `a + (b + c)`

Оба варианта дадут один и тот же результат. Ассоциативность позволяет программисту выбирать порядок выполнения операций, что может улучшить читаемость кода.

### Поток команд

Поток команд - это последовательность команд, которые исполняются в определённом порядке. В языках программирования поток команд определяется с помощью инструкций и циклов.

Например, следующий фрагмент кода:

```python
x = 5
y = x * 2
print(y)
```

Имеет следующий поток команд:

1. Инициализация переменной `x` значением `5`.
2. Выполнение выражения `x * 2`, которое присваивает значение `10` переменной `y`.
3. Вывод значения `y` в консоль.

### Оператор if

Оператор if - это инструкция, которая проверяет условие и выполняет определённый блок кода, если условие истинно.

Например:

```python
x = 5
if x > 10:
    print("x больше 10")
```

В этом примере оператор `if` проверяет, равно ли значение `x` больше `10`. Если это условие истинно, то блок кода `print("x больше 10")` будет выполнен.

Оператор if может иметь следующие формы:

- `if условие:`
- `if условие: блок_кода`

### Оператор for

Оператор for - это инструкция, которая выполняет определённый блок кода для каждого элемента в коллекции (например, массиве или строке).

Например:

```python
fruits = ["яблоко", "банан", "груша"]
for fruit in fruits:
    print(fruit)
```

В этом примере оператор `for` проходит по каждому элементу в коллекции `fruits`, присваивая значение каждого элемента переменной `fruit`. Затем блок кода `print(fruit)` выполняется для каждого значения `fruit`.

Оператор for может иметь следующие формы:

- `for переменная in коллекция:`
- `for переменная in range(начало, конец):`

### Оператор while

Оператор while - это инструкция, которая повторяет определённый блок кода, пока условие не станет ложным.

Например:

```python
x = 0
while x < 5:
    print(x)
    x += 1
```

В этом примере оператор `while` проверяет, равно ли значение `x` меньше `5`. Если это условие истинно, то блок кода `print(x)` и `x += 1` будет выполнен. Этот процесс повторяется, пока значение `x` не станет больше или равным `5`.

Оператор while может иметь следующие формы:

- `while условие:`
- `while условие: блок_кода`
