## Основные концепции

**Основные концепции: Списки и кортежи**

### Списки

Списки - это тип данных, который позволяет хранить коллекцию элементов в определённом порядке. В списке каждый элемент имеет индекс или номер, что позволяет обращаться к нему по имени.

Например:
```python
fruits = ['яблоко', 'банан', 'груша']
print(fruits[0])  # выводит: яблоко
```
В этом примере мы создали список `fruits`, в котором хранятся три элемента. Мы можем обращаться к каждому элементу по его индексу, который начинается с нуля.

Списки могут содержать разные типы данных, не только строки:
```python
numbers = [1, 2, 3]
print(numbers[0])  # выводит: 1
```
### Кортежи

Кортежи - это другой тип данных, который также позволяет хранить коллекцию элементов. Однако в кортеже каждый элемент фиксирован и не может быть изменен.

Например:
```python
colors = ('красный', 'зелёный', 'синий')
print(colors[0])  # выводит: красный
```
В этом примере мы создали кортеж `colors`, в котором хранятся три элемента. Мы можем обращаться к каждому элементу по его индексу, но не можем изменить их.

Кортежи также могут содержать разные типы данных:
```python
numbers = (1, 2, 3)
print(numbers[0])  # выводит: 1
```
### Структуры данных

Структура данных - это общее понятие, которое относится к способам хранения и организации данных в компьютерной программе. Включает в себя различные типы данных, такие как списки, кортежи, словари и множества.

Например:
```python
fruits = {'яблоко': 5, 'банан': 10}
print(fruits['яблоко'])  # выводит: 5
```
В этом примере мы создали словарь `fruits`, в котором хранятся два элемента. Мы можем обращаться к каждому элементу по его ключу.

### Пакеты

Пакет - это коллекция модулей, которые связаны между собой и могут быть использованы вместе. Пакеты позволяют организовать код и делают его более управляемым.

Например:
```python
import math
print(math.pi)  # выводит: 3.14159...
```
В этом примере мы импортировали пакет `math`, который содержит функции для работы с математическими операциями.

### Модули

Модуль - это отдельная единица кода, которая может быть использована в других программах. Модули позволяют разделить код на логически связанные части и делают его более управляемым.

Например:
```python
import my_module
print(my_module.my_function())  # выводит: результат функции
```
В этом примере мы импортировали модуль `my_module`, который содержит функцию `my_function`. Мы можем использовать эту функцию в нашей программе.

### Объекты

Объект - это экземпляр класса, который имеет свои собственные свойства и методы. Объекты позволяют организовать код и делают его более управляемым.

Например:
```python
class Car:
    def __init__(self, color):
        self.color = color

my_car = Car('красный')
print(my_car.color)  # выводит: красный
```
В этом примере мы создали класс `Car`, который имеет свойство `color`. Мы создали объект `my_car` из этого класса и можем обращаться к его свойствам.

### Классы

Класс - это шаблон, который определяет свойства и поведение объекта. В Python классы используются для создания пользовательских типов данных.

Например, мы можем определить класс `Person` с атрибутами `имя` и `возраст`, а также методом `приветствие()`:

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greeting(self):
        print(f"Привет, меня зовут {self.name} и мне {self.age} лет.")

# Создаем экземпляр класса Person
person = Person("Иван", 30)
# Вызываем метод greeting()
person.greeting()  # Выводит: Привет, меня зовут Иван и мне 30 лет.
```

### Методы строк

Методы строк - это функции, которые можно вызывать на строках. Они используются для манипулирования строками.

Например, мы можем использовать метод `upper()` для преобразования строки в верхний регистр:

```python
строка = "hello"
print(строка.upper())  # Выводит: HELLO

# Используем метод replace() для замены символов:
строка = "hello world"
print(строка.replace("world", "Python"))  # Выводит: hello Python
```

### Индексация

Индексация - это способ доступа к элементам списка или кортежа по их индексу.

Например, мы можем использовать индекс `0` для доступа к первому элементу списка:

```python
список = ["apple", "banana", "cherry"]
print(список[0])  # Выводит: apple

# Используем индекс -1 для доступа к последнему элементу:
print(список[-1])  # Выводит: cherry
```

### Срезы

Срезы - это способ доступа к части списка или кортежа.

Например, мы можем использовать срез `1:3` для доступа к второму и третьему элементам списка:

```python
список = ["apple", "banana", "cherry"]
print(список[1:3])  # Выводит: ['banana', 'cherry']

# Используем срез :2 для доступа ко всем элементам до второго:
print(список[:2])  # Выводит: ['apple', 'banana']
```

### Zip

Zip - это функция, которая объединяет два или более списков в один список кортежей.

Например, мы можем использовать `zip()` для объединения двух списков:

```python
список1 = ["apple", "banana", "cherry"]
список2 = [1, 2, 3]
print(list(zip(список1, список2)))  # Выводит: [('apple', 1), ('banana', 2), ('cherry', 3)]
```

### Self

Self - это специальный аргумент в методах классов, который ссылается на экземпляр класса.

Например, мы можем использовать `self` для доступа к атрибутам класса:

```python
class Person:
    def __init__(self, name):
        self.name = name

    def greeting(self):
        print(f"Привет, меня зовут {self.name}.")

person = Person("Иван")
person.greeting()  # Выводит: Привет, меня зовут Иван.
```

### Объектно-ориентированное программирование

Объектно-ориентированное программирование (ООП) — это парадигма программирования, которая основана на представлении программного кода как набора взаимодействующих объектов. Каждый объект имеет свои собственные свойства и поведение, что позволяет писать более модульный и гибкий код.

В ООП объекты представляют собой экземпляры классов, которые определяют их структуру и поведение. Классы — этоblueprints для создания объектов, они содержат в себе описания свойств и методов, которые будут доступны для каждого объекта.

Например, мы можем представить автомобиль как объект с собственными свойствами (цвет, модель, год выпуска) и поведением (двигаться, останавливаться). Класс «Автомобиль» будет определять эти свойства и методы, а затем мы сможем создать несколько экземпляров этого класса, каждый из которых будет иметь свои собственные значения свойств.

### Поля

Поля — это переменные, которые объявляются в классе и принадлежат каждому объекту этого класса. Они хранят данные о свойствах объекта и могут быть доступны только внутри класса или через методы класса.

Например, у нас есть класс «Автомобиль», который имеет поля: «цвет», «модель» и «год выпуска». Каждый экземпляр этого класса будет иметь свои собственные значения этих полей.

```python
class Автомобиль:
    def __init__(self, цвет, модель, год_выпуска):
        self.цвет = цвет
        self.модель = модель
        self.год_выпуска = год_выпуска

автомобиль1 = Автомобиль('красный', 'Toyota', 2015)
автомобиль2 = Автомобиль('синий', 'Honda', 2020)

print(автомобиль1.цвет)  # красный
print(автомобиль2.модель)  # Honda
```

### Атрибуты

Атрибуты — это значения полей объекта, которые хранят данные о свойствах объекта. Они могут быть доступны только внутри класса или через методы класса.

Например, у нас есть класс «Автомобиль», который имеет атрибуты: «цвет», «модель» и «год выпуска». Каждый экземпляр этого класса будет иметь свои собственные значения этих атрибутов.

```python
class Автомобиль:
    def __init__(self, цвет, модель, год_выпуска):
        self.цвет = цвет
        self.модель = модель
        self.год_выпуска = год_выпуска

автомобиль1 = Автомобиль('красный', 'Toyota', 2015)
автомобиль2 = Автомобиль('синий', 'Honda', 2020)

print(автомобиль1.get_цвет())  # красный
print(автомобиль2.get_модель())  # Honda
```

### Переменные экземпляра

Переменные экземпляра — это переменные, которые объявляются в классе и принадлежат каждому объекту этого класса. Они хранят данные о свойствах объекта и могут быть доступны только внутри класса или через методы класса.

Например, у нас есть класс «Автомобиль», который имеет переменные экземпляра: «цвет», «модель» и «год выпуска». Каждый экземпляр этого класса будет иметь свои собственные значения этих переменных.

```python
class Автомобиль:
    def __init__(self, цвет, модель, год_выпуска):
        self.цвет = цвет
        self.модель = модель
        self.год_выпуска = год_выпуска

автомобиль1 = Автомобиль('красный', 'Toyota', 2015)
автомобиль2 = Автомобиль('синий', 'Honda', 2020)

print(автомобиль1.цвет)  # красный
print(автомобиль2.модель)  # Honda
```

### Переменные класса

Переменные класса — это переменные, которые объявляются в классе и принадлежат самому классу. Они хранят данные о свойствах класса и могут быть доступны для всех объектов этого класса.

Например, у нас есть класс «Автомобиль», который имеет переменную класса: «количество_автомобилей». Каждый экземпляр этого класса будет иметь доступ к этой переменной.

```python
class Автомобиль:
    количество_автомобилей = 0

    def __init__(self, цвет, модель, год_выпуска):
        self.цвет = цвет
        self.модель = модель
        self.год_выпуска = год_выпуска
        Автомобиль.количество_автомобилей += 1

автомобиль1 = Автомобиль('красный', 'Toyota', 2015)
автомобиль2 = Автомобиль('синий', 'Honda', 2020)

print(Автомобиль.количество_автомобилей)  # 2
```
