### `join()`

Метод `join()` используется для объединения элементов списка или кортежа в одну строку. Он принимает на вход iterable (например, список или кортеж) и возвращает строку, в которой все элементы из этого iterable соединены между собой.

Например:
```python
fruits = ['apple', 'banana', 'cherry']
print(', '.join(fruits))  # Выводит: apple, banana, cherry
```
В этом примере метод `join()` используется для объединения элементов списка `fruits` в одну строку с разделителем `,`.

### `replace()`

Метод `replace()` используется для замены одного или нескольких символов в строке на другой символ или строку. Он принимает на вход два аргумента: исходную строку и строку, которую нужно заменить.

Например:
```python
text = 'Hello, world!'
print(text.replace('world', 'Python'))  # Выводит: Hello, Python!
```
В этом примере метод `replace()` используется для замены слова «world» на «Python» в строке `text`.

Python поддерживает различные кодировки, включая UTF-8, UTF-16 и т. д. Чтобы указать кодировку для строки, можно использовать атрибут `encoding`:
