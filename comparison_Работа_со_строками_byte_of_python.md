Работа со строками является фундаментальной частью программирования на Python. В этом разделе мы рассмотрим основы работы с текстом в языке Python, включая литеральные константы, числа, строки и переменные.

**Литеральные константы**

Первым типом данных, который мы рассматриваем, являются литеральные константы. Литеральной константой называется значение, которое используется буквально в программе без каких-либо изменений. Примерами литеральных констант могут быть числа, такие как 5 или 1.23, а также строки, такие как 'Это строка' или "It's a string!". # [СТРАНИЦА 36]

**Числа**

В Python существуют три типа чисел: целые, числа с плавающей точкой и комплексные числа.

* Целыми числами являются те, которые не содержат десятичной точки. Примером целого числа может быть 2.
* Числами с плавающей точкой (или «плавающими» для краткости) могут быть 3.23 или 52.3E-4. Обозначение E показывает степень числа 10. В данном случае 52.3E-4 означает 52,3 × 10^(-4).
* Комплексными числами являются те, которые содержат мнимую часть, обозначаемую буквой «j». Примерами комплексных чисел могут быть (-5+4j) или (2.3 - 4.6j). # [СТРАНИЦА 37]

**Строки**

Строкой называется последовательность символов. Чаще всего строки представляют собой набор слов. Слова могут быть на английском языке, а также на любом другом языке, поддерживаемом стандартом Unicode, что означает практически на любом языке мира.

Замечание для опытных программистов: В Python 3 нет ASCII-строк, поскольку Unicode является надмножеством (включает в себя) ASCII. Если необходимо получить строку строго в кодировке ASCII, используйте str.encode("ascii"). Подробнее можно посмотреть на обсуждении этого вопроса на StackOverflow.

По умолчанию все строки представляют собой Unicode. # [СТРАНИЦА 38]

* Одинарными кавычками строку можно указать так: 'Фраза в кавычках'. Всякий пробел или табуляция сохраняется как есть.

### Описание строк как объектов Python и их операций

В Python строки являются объектами класса `str`, которые можно создавать с помощью кавычек или двойных кавычек. Например:
```
# Пример со страницы 36

print(name)
```
Строки в Python поддерживают различные операции, такие как индексация, срезы и методы. Индексация позволяет получить доступ к конкретному символу строки по его номеру индекса. Например:
```
# Пример со страницы 37
name = 'Swaroop'
print(name[0])  # Выводит 'S'
```
Срезами можно вырезать часть строки, начиная с определенного индекса и заканчивая другим индексом. Например:
```
# Пример со страницы 37

print(name[0:5])  # Выводит 'Swaro'
```
Методами можно выполнять различные операции над строкой, такие как преобразование в верхний или нижний регистр, удаление пробелов и т. д. Например:
```
# Пример со страницы 38

```
### Объяснение работы с шаблонами и регулярными выражениями

Шаблоны и регулярные выражения используются для поиска и замены текста в строке. Шаблоном можно считать строку, которая содержит специальные символы, которые указывают на то, где следует искать или заменять текст.

Регулярное выражение — это шаблон, который используется для поиска и замены текста в строке с помощью функции `re`. Например:
```
# Пример со страницы 40

text = 'Hello, world!'
print(re.search('world', text))  # Выводит <_sre.SRE_Match object at ...>
```
Функция `re` позволяет выполнять различные операции с шаблоном, такие как поиск, замена и удаление.

### Применение методов строк

Методы строк можно использовать для выполнения различных операций над строкой, таких как:

* Преобразование в верхний или нижний регистр
* Удаление пробелов
* Замена текста
