**Введение**

В этой лекции мы будем изучать строки и форматирование в Python. Строки - это последовательности символов, которые могут содержать текст или числа. Форматирование - это способ преобразования строк в желаемый вид.

**Основные концепции**

Строки в Python представляются как объекты класса `str`. Они могут быть созданы с помощью функции `print()`, которая выводит строку на экран.

```python
# Пример со страницы 36
>>> print('hello world')
hello world
```

Строки можно объединять с помощью оператора `+`.

```python
# Пример со страницы 37
s1 = 'Hello'
s2 = 'World'
print(s1 + ' ' + s2)
```

Форматирование строк позволяет преобразовать их в желаемый вид. Например, можно вывести строку с определенным форматом.

```python
# Пример со страницы 38
name = 'Swaroop'
age = 25
print('%s is %d years old.' % (name, age))
```

**Примеры кода**

1. Создание строки:

```python
# Пример со страницы 36
>>> print('hello world')
hello world
```

2. Объединение строк:

```python
# Пример со страницы 37

s2 = 'World'
print(s1 + ' ' + s2)
```

3. Форматирование строки:

```python
# Пример со страницы 38
name = 'Swaroop'
age = 25
print('%s is %d years old.' % (name, age))
```

**Практические советы**

* Используйте функцию `print()` для вывода строк на экран.
* Объединяйте строки с помощью оператора `+`.
* Используйте форматирование строк для преобразования их в желаемый вид.
