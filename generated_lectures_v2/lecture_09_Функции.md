**Введение**

В этой лекции мы будем изучать фундаментальные концепции управления потоком команд в Python. Мы рассмотрим операторы if, while и for, а также функции, которые являются важнейшим инструментом для организации кода.

**Основные концепции**

### Оператор if

Оператор if используется для проверки условий. Если условие верно, выполняется блок команд (if-блок), в противном случае выполняется другой блок команд (else-блок). Блок else является не обязательным.

Пример:
```python
# Со страницы 59
number = 23
guess = int(input('Введите целое число : '))
if guess == number:
    print('Поздравляю, вы угадали,')
    print('(хотя и не выиграли никакого приза!)')
elif guess < number:
    print('Нет, загаданное число немного больше этого.')
else:
    print('Нет, загаданное число немного меньше этого.')
```

### Оператор while

Оператор while позволяет многократно выполнять блок команд до тех пор, пока не будет выполнено некоторое условие. Это один из операторов цикла.

Пример:
```python
# Со страницы 62
number = 23
running = True
while running:
    guess = int(input('Введите целое число : '))
    if guess == number:
        print('Поздравляю, вы угадали.')
        running = False
    elif guess < number:
        print('Нет, загаданное число немного больше этого')
    else:
        print('Нет, загаданное число немного меньше этого.')
```

### Оператор for

Оператор for..in используется для итерации по последовательности объектов. Он позволяет многократно выполнять блок команд для каждого элемента в последовательности.

Пример:
```python
# Со страницы 63
for i in range(1, 5):
    print(i)
else:
    print('Цикл for закончен')
```

### Оператор break

Оператор break используется для прерывания цикла. Если циклы for или while прервать оператором break, соответствующие им блоки else не будут выполняться.

Пример:
```python
# Со страницы 65
while True:
    s = input('Введите что-нибудь : ')
    if s == 'выход':
        break
    print('Длина строки: ', len(s))
```

### Оператор continue

Оператор continue используется для указания Python, что необходимо пропустить все оставшиеся команды в текущем блоке цикла и продолжить с следующей итерацией.

Пример:
```python
# Со страницы 66
while True:
    s = input('Введите что-нибудь : ')
    if s == 'выход':
        break
    if len(s) < 3:
        print('Слишком мало')
        continue
    print('Введённая строка достаточной длины')
```

### Функции

Функция — это многократно используемый фрагмент программы. Она позволяет дать имя определённому блоку команд, чтобы в последствии запускать этот блок по указанному имени в любом месте программы.

Пример:
```python
# Со страницы 68
def sayHello():
    print('Привет, Мир!')
sayHello()
```

### Параметры функций

Функции могут принимать параметры — некоторые значения, которые передаются функции для того, чтобы она что-либо сделала с ними.

Пример:
```python
# Со страницы 69
def printMax(a, b):
    if a > b:
        print(a, 'максимально')
    elif a == b:
        print(a, 'равно', b)
    else:
        print(b, 'максимально')
printMax(3, 4)
```

### Локальные переменные

При объявлении переменных внутри определения функции они не связаны с другими переменными с таким же именем за пределами функции.

Пример:
```python
# Со страницы 70
x = 50
def func(x):
    print('x равен', x)
    x = 2
    print('Замена локального x на', x)
func(10)
```

### Зарезервированное слово «global»

Чтобы присвоить некоторое значение переменной, определённой на высшем уровне программы (т.е. не в какой-либо области видимости), необходимо указать Python, что это имя является глобальным.

Пример:
```python
# Со страницы 71
x = 50
def func():
    global x
    print('x равно', x)
    x = 2
    print('Заменяем глобальное значение x на', x)
func()
```
