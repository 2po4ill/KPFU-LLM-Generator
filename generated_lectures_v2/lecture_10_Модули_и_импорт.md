**Модули и импорт**

В этом разделе мы рассмотрим основные концепции модулей и импорта в Python. Модули — это файлы с расширением `.py`, содержащие функции, переменные и классы, которые можно использовать в других программах.

**Основные концепции**

* **Модуль**: Файл с расширением `.py`, содержащий функции, переменные и классы.
* **Импорт**: Процесс использования функций, переменных и классов из модуля в другой программе.
* **Стандартная библиотека Python**: Сборник модулей, которые поставляются с интерпретатором Python.

**Примеры кода**

```python
# Пример со страницы 71
import sys
print('Аргументы командной строки:')
for i in sys.argv:
    print(i)
```

```python
# Пример со страницы 72
def say(message, times = 1):
    print(message * times)
say('Привет')
say('Мир', 5)
```

```python
# Пример со страницы 73
def func(a, b=5, c=10):
    print('a равно', a, ', b равно', b, ', а c равно', c)
func(3, 7)
func(25, c=24)
func(c=50, a=100)
```
