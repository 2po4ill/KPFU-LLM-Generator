**Основные концепции**

В языке Python существует два основных способа работы: интерактивный режим и работа с файлами. В интерактивном режиме мы можем писать код прямо в командной строке, а затем запускать его. Однако это не самый удобный способ работать с программами, поскольку нам нужно каждый раз запускать Python и вводить наш код.

Чтобы решить эту проблему, мы можем сохранять наши программы в файлах с расширением `.py`. Затем мы можем использовать редактор для написания и редактирования наших программ. В этом случае мы можем запускать программы прямо из файла, не нужно каждый раз запускать Python.

**Примеры кода**

### Пример 1: Первая программа на Python

```python
# [СТРАНИЦА 38]
print('Привет, Мир!')
```

Этот пример демонстрирует, как можно написать первую программу на Python. Мы используем функцию `print` для вывода текста на экран.

### Пример 2: Использование hashbang

```python
# [СТРАНИЦА 40]
#!/usr/bin/env python3
print('Привет, Мир!')
```

Этот пример демонстрирует, как можно использовать hashbang для указания интерпретатора Python. Это позволяет нам запускать программы напрямую из файла, не нужно каждый раз запускать Python.

### Пример 3: Работа с переменными

```python
# [СТРАНИЦА 42]
x = 5
y = x * 2
print(y)
```

Этот пример демонстрирует, как можно использовать переменные в программе. Мы присваиваем значение `5` переменной `x`, а затем используем ее для вычисления значения `y`.

**Практические советы**

* Используйте редактор с подсветкой синтаксиса, чтобы сделать написание и редактирование программ более удобным.
* Используйте hashbang для указания интерпретатора Python, чтобы можно было запускать программы напрямую из файла.
* Используйте переменные для хранения значений и вычислений в программе.
