**Введение**

Работа со строками является важнейшим аспектом программирования на Python. В этой лекции мы рассмотрим основные концепции, связанные с работой со строками, и покажем, как использовать их в своих программах.

Сначала давайте разберемся, что такое строки. Строка – это последовательность символов. Чаще всего строки представляют собой набор слов или фраз на языке программирования. В Python строки можно представлять как литеральные константы, которые не могут быть изменены после создания.

Например: `print('Привет, Мир!')` – это строка, которая содержит несколько символов и представляет собой фразу на русском языке.

В Python 3 нет ASCII-строк, поскольку Unicode является надмножеством (включает в себя) ASCII. Если необходимо получить строку строго в кодировке ASCII, можно использовать метод `str.encode("ascii")`.

По умолчанию все строки в Python являются объектами класса str. Этот класс имеет множество полезных методов, которые могут быть использованы для работы со строками.

Например: `name = 'Swaroop'` – это строка, которая является объектом класса str. Мы можем использовать методы этого класса для проверки части строки или удаления краевых пробелов.

Метод `startswith` используется для определения, начинается ли строка с заданной подстроки. Оператор `in` используется для проверки, является ли данная строка частью другой строки. Метод `find` используется для определения позиции заданной подстроки в строке.

Например: `if name.startswith('Swa'): print('Да, строка начинается на "Swa"')`

Мы можем также использовать метод `join` для объединения элементов последовательности с указанной строкой в качестве разделителя между элементами. Этот метод возвращает большую строку, сгенерированную таким образом.

Например: `delimiter = '_*_'; mylist = ['Бразилия', 'Россия', 'Индия', 'Китай']; print(delimiter.join(mylist))`

**Основные концепции**

В этом разделе мы рассмотрим основные концепции Python, которые являются фундаментальными для понимания языка. Мы начнем с литеральных констант и их типов, а затем перейдем к понятию строки как последовательности символов и ее свойствах.

**7.1 Литеральные константы**

Литеральные константы — это значения, которые используются напрямую в программе без необходимости дополнительной обработки. Примерами литеральных констант могут быть числа, такие как 5 или 3.14, а также строки, такие как "Привет, Мир!" или 'Это строка'. Литеральные константы называются так потому, что они представляют собой буквальное значение, которое используется напрямую в программе.

**7.2 Типы литеральных констант**

Литеральные константы могут иметь разные типы. Например:

* Целые числа: 5, -3, 0
* Числа с плавающей точкой (или "плавающие" для краткости): 3.14, -0.5, 2.5E-4
* Комплексные числа: (-5+4j), (2.3 - 4.6j)

**7.3 Строки**

Строка — это последовательность символов. Чаще всего строки представляют собой набор слов. Слова могут быть на английском языке или на любом другом языке, поддерживаемом стандартом Unicode.

Строки в Python можно обозначать двумя способами: с использованием одинарных кавычек ('') или двойных кавычек (""). Например:

* 'Фраза в кавычках'
* "What's your name?"

**7.4 Строки неизменяемы**

Строки в Python являются неизменяемыми, что означает, что после создания строки ее нельзя изменить.

**7.5 Объединение строковых констант**

Если расположить рядом две строковые константы, Python автоматически их объединит. Например:

* 'What\'s ' 'your name?' -> "What's your name?"

**7.6 Метод format**

Метод format() позволяет создавать строки на основе шаблона и значений. Например:

* age = 26
* name = 'Swaroop'
* print('Возраст {0} -- {1} лет.'.format(name, age)) -> "Возраст Swaroop -- 26 лет."

**7.7 Переменные**

Переменные — это области памяти компьютера, в которых хранится некоторая информация. В отличие от констант, к переменной нужно получить доступ, чтобы изменить ее значение.

Имена идентификаторов должны начинаться с буквы или символа подчеркивания ("_"). Остальная часть имени может состоять из букв, знаков подчеркивания и цифр. Имена идентификаторов чувствительны к регистру, что означает, что "myname" и "MyName" — это разные переменные.

**7.8 Присваивание**

Присваивание — это операция, которая присваивает значение одной переменной другой переменной. Например:

* i = 5
* print(i) -> 5

**7.9 Логические и физические строки**

Логическая строка — это то, что Python видит как единое предложение. Физическая строка — это то, что вы видите в редакторе. Например:

* print('Привет, Мир!') -> логическая строка
* print('Привет, Мир!\n' 'Это вторая строка.') -> физическая строка

**7.10 Отступы**

Отступы — это пробелы в начале строки. Они важны в Python и используются для определения начала логической строки.

### Пример 3. Создание многострочной строки
