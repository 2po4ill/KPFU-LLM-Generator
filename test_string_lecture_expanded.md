**Работа со строками: форматирование, методы строк, срезы**

**Введение**

В этом разделе мы рассмотрим основные концепции работы со строками в Python. Строки являются одной из наиболее часто используемых типов данных в программировании, и понимание их свойств и методов является важнейшим навыком для любого программиста.

**Основные концепции**

1. **Форматирование строк**: В Python существует несколько способов форматировать строки. Одним из наиболее простых способов является использование функции `format()` или метода `__format__()`. Например, можно использовать следующий код:
```python
# Пример со страницы 102
name = "Светлана"
age = 30
print("Меня зовут {} и мне {}".format(name, age))
```
2. **Методы строк**: Строки в Python имеют множество методов, которые можно использовать для различных операций. Например, метод `upper()` преобразует строку в верхний регистр:
```python
# Пример со страницы 103
name = "Светлана"
print(name.upper())
```
3. **Срезы**: Срезы являются еще одним важным аспектом работы со строками. Они позволяют получить часть строки, начиная с определенного индекса и заканчивая другим индексом:
```python
# Пример со страницы 104
name = "Светлана"
print(name[0:5])
```
**Примеры кода**

1. **Форматирование строки**
```python
# Пример со страницы 102

```
2. **Использование методов строк**
```python
# Пример со странице 103

print(name.upper())
```
3. **Срезы**
```python
# Пример со странице 104

```

**Практические советы**

1. Используйте функцию `format()` или метод `__format__()`, чтобы форматировать строки.
2. Познакомьтесь с основными методами строк, такими как `upper()`, `lower()`, `split()`, и т.д.
3. Используйте срезы для получения части строки.
